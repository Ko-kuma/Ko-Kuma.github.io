파이썬에서 가장 많이 사용되는 자료형 중 하나인 **문자열(String)**에 대해 상세히 정리해봅니다. 문자열의 생성, 인덱싱의 원리, 그리고 다양한 포매팅 방법까지 빈틈없이 알아봅시다.

## 1. 문자열 만들기 (4가지 방법)

파이썬에서 문자열을 만드는 방법은 총 4가지가 있어, 상황에 따라 유연하게 사용할 수 있습니다.

1.  **큰따옴표(`"`)로 양쪽 둘러싸기**: `"Hello World"`
2.  **작은따옴표(`'`)로 양쪽 둘러싸기**: `'Python is fun'`
3.  **큰따옴표 3개(`"""`)로 양쪽 둘러싸기**: `"""Life is too short"""`
4.  **작은따옴표 3개(`'''`)로 양쪽 둘러싸기**: `'''You need python'''`

### 문자열 안에 따옴표 포함시키기
문자열 내부에 따옴표를 문자로 표현하고 싶을 때는 다음과 같이 처리합니다.

* **다른 따옴표 사용**: 문자열 전체를 큰따옴표로 감싸고 내부에 작은따옴표 사용 (반대도 가능)
    ```python
    say = "He said, 'Python is easy'"  # 큰따옴표 안에 작은따옴표
    food = 'Python\'s favorite food'   # (역슬래시 이용)
    ```
* **역슬래시(`\`) 사용**: 따옴표 바로 앞에 `\`를 붙이면 문자로 인식
    ```python
    say = "\"Python is very easy.\" he says."
    ```

## 2. 여러 줄 문자열과 이스케이프 코드

### 여러 줄 문자열 (Multi-line String)
여러 줄에 걸친 문자열을 변수에 대입하고 싶을 때는 **이스케이프 코드(`\n`)**를 쓰거나 **따옴표 3개**를 사용합니다.

```python
# 방법 1: 이스케이프 코드 사용 (가독성이 떨어질 수 있음)
multiline = "Life is too short\nYou need python"

# 방법 2: 따옴표 3개 사용 (추천)
multiline = '''
Life is too short
You need python
'''

```

### 이스케이프 코드(Escape Code) 정리

프로그래밍에서 사용할 수 있도록 미리 정의해 둔 문자 조합입니다.

| 코드 | 설명                         |
| ---- | ---------------------------- |
| `\n` | 줄 바꿈 (New line)           |
| `\t` | 탭 (Tab) 간격 띄우기         |
| `\\` | 역슬래시(`\`) 문자 자체 표현 |
| `\'` | 작은따옴표 문자 표현         |
| `\"` | 큰따옴표 문자 표현           |

## 3. 문자열 연산

파이썬에서는 문자열도 더하거나 곱할 수 있습니다.

* **더하기(`+`)**: 문자열 연결 (Concatenation)
* **곱하기(`*`)**: 문자열 반복
* **길이 구하기(`len()`)**: 문자열의 길이 반환

```python
head = "Python"
tail = " is fun!"

print(head + tail)  # "Python is fun!"
print(head * 2)     # "PythonPython"
print(len(head))    # 6

```

## 4. 인덱싱(Indexing)과 슬라이싱(Slicing)

문자열은 각 문자마다 고유한 번호(인덱스)가 매겨져 있어 특정 부분을 가리키거나 잘라낼 수 있습니다.

### 1) 인덱싱 (Indexing): 0부터 시작!

**[중요]** 파이썬은 숫자를 **0부터** 셉니다. (Zero-based Indexing)
따라서 첫 번째 문자의 번호는 `1`이 아니라 **`0`**입니다.

**[문자열 "Life"의 인덱스 구조]**

| 문자       | L     | i     | f     | e     |
| ---------- | ----- | ----- | ----- | ----- |
| **인덱스** | **0** | **1** | **2** | **3** |
| -인덱스    | -4    | -3    | -2    | -1    |

* **양수 인덱스**: 앞에서부터 0, 1, 2... 순서로 셉니다.
* **음수 인덱스**: 뒤에서부터 셀 때는 -1부터 시작합니다. (뒤에서 첫 번째는 -1)

```python
a = "Life is too short"

print(a[0])   # 'L' (첫 번째 문자) -> 0번!
print(a[3])   # 'e' (네 번째 문자) -> 3번!
print(a[-1])  # 't' (가장 마지막 문자)

```

### 2) 슬라이싱 (Slicing)

범위를 지정해서 문자열의 일부를 추출합니다.

* **문법**: `변수명[시작번호 : 끝번호]`
* **주의**: **끝번호에 해당하는 문자는 포함되지 않습니다.** (끝번호 - 1 까지만 가져옴)

```python
a = "Life is too short"

print(a[0:4])   # 'Life' (0, 1, 2, 3번까지 추출. 4번은 포함 X)
print(a[12:])   # 'short' (12번부터 끝까지)
print(a[:4])    # 'Life' (처음부터 3번까지)
print(a[:])     # 전체 문자열 복사

```

## 5. [중요] 문자열은 '불변(Immutable)'이다

파이썬의 문자열 자료형은 **생성된 후 그 값을 변경할 수 없습니다.** (Immutable)

```python
a = "Pithon"
# a[1] = 'y'  # -> 오류 발생 (TypeError)

```

오류를 피하고 싶다면 슬라이싱 등을 이용해 **새로운 문자열을 만들어야** 합니다.

```python
a = "Pithon"
new_a = a[:1] + 'y' + a[2:]
print(new_a)  # Python

```

## 6. 문자열 포매팅 (Formatting)

문자열 안에 특정한 값(변수)을 바꿔가며 넣고 싶을 때 사용합니다.

### 1) 포맷 코드 (%) 사용

C언어 스타일의 포매팅 방식입니다.

| 코드 | 설명                                          |
| ---- | --------------------------------------------- |
| `%d` | 정수 (Integer)                                |
| `%s` | 문자열 (String) - *어떤 값이든 문자열로 변환* |
| `%f` | 부동 소수 (Floating-point)                    |

```python
# 1. 숫자/문자열 바로 대입
print("I eat %d apples." % 3)
print("I eat %s apples." % "five")

# 2. 2개 이상 값 넣기
number = 10
day = "three"
print("I ate %d apples. so I was sick for %s days." % (number, day))

# 3. 정렬과 소수점
print("%10s" % "hi")    # "        hi" (오른쪽 정렬)
print("%-10s" % "hi")   # "hi        " (왼쪽 정렬)
print("%0.4f" % 3.42134234) # "3.4213" (소수점 4자리)

```

### 2) format() 함수 사용

중괄호 `{}`를 사용하는 방식입니다.

```python
# 기본 사용
print("I eat {0} apples".format(3))
print("number: {0}, day: {1}".format(10, "three"))

# 이름으로 넣기
print("I ate {number} apples. so I was sick for {day} days.".format(number=10, day=3))

# 정렬 및 공백 채우기
print("{0:<10}".format("hi"))  # 왼쪽 정렬 ("hi        ")
print("{0:>10}".format("hi"))  # 오른쪽 정렬 ("        hi")
print("{0:^10}".format("hi"))  # 가운데 정렬 ("    hi    ")
print("{0:=^10}".format("hi")) # 공백 채우기 ("====hi====")

# 소수점 표현
y = 3.42134234
print("{0:0.4f}".format(y))    # 3.4213
print("{{ and }}".format())    # { and } (중괄호 자체 표현)

```

### 3) f-string 포매팅 (추천)

파이썬 3.6+ 부터 지원하며 가장 직관적이고 빠릅니다. 문자열 앞에 `f`를 붙입니다.

```python
name = "홍길동"
age = 30

# 변수 그대로 사용
print(f'나의 이름은 {name}입니다. 나이는 {age}입니다.')

# 표현식 사용 (연산 가능)
print(f'내년에는 {age + 1}살이 됩니다.')

# 정렬과 소수점
print(f'{"hi":<10}')  # 왼쪽 정렬
print(f'{"hi":>10}')  # 오른쪽 정렬
print(f'{3.42134234:0.4f}') # 소수점 4자리

```

## 7. 문자열 관련 함수 (Method)

문자열 자료형이 자체적으로 가지고 있는 유용한 함수들입니다.

### 문자 개수 세기 / 위치 찾기

* `count('x')`: 문자 'x'의 개수 반환
* `find('x')`: 문자 'x'가 처음 나온 위치 반환 (**없으면 -1 반환**)
* `index('x')`: 문자 'x'가 처음 나온 위치 반환 (**없으면 오류 발생**)

```python
a = "hobby"
print(a.count('b')) # 2
print(a.find('b'))  # 2
print(a.find('k'))  # -1 (없음)

```

### 문자열 조작하기

* `join()`: 문자열 사이에 문자 삽입
* `upper()` / `lower()`: 대문자로 / 소문자로 변환
* `lstrip()` / `rstrip()` / `strip()`: 공백 제거 (좌 / 우 / 양쪽)
* `replace(old, new)`: 문자열 바꾸기
* `split()`: 문자열 나누기 (리스트로 반환)

```python
a = ","
print(a.join('abcd')) # "a,b,c,d"

b = " hi "
print(b.strip())      # "hi"

c = "Life is too short"
print(c.replace("Life", "Your leg")) # "Your leg is too short"
print(c.split())      # ['Life', 'is', 'too', 'short']

```
