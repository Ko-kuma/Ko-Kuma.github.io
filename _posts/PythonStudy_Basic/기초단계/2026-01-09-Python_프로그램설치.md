# 실행 환경 설정과 파이썬 실행 방법
파이썬 코드를 실행하려면 먼저 파이썬이 설치되어 있어야 하며,  
터미널(명령 프롬프트) 또는 VS Code와 같은 실행 환경을 이해해야 합니다.

이 글에서는 **설치 → 설치 확인 → 실행 방식(REPL, 파일 실행) → VS Code 설정** 순서로 정리합니다.

---
## 시작 전 기본 용어

본격적인 설치에 앞서, 앞으로 계속 보게 될 화면과 용어부터 간단히 잡고 가겠습니다.

- **터미널(Terminal) / 콘솔(Console):** 마우스 대신 **글자(명령어)로 컴퓨터에게 일을 시키는 검은색 창**입니다.
- **명령(Command):** 터미널에 입력하는 **"컴퓨터야, 이거 해줘"**라는 문장입니다.
- **프롬프트(Prompt):** 터미널에서 명령 입력을 기다리며 깜빡이는 표시입니다. (예: `$`, `>`, `>>>`)

---

## 1. macOS에서 파이썬 설치

macOS에는 파이썬이 이미 설치되어 있을 수도 있지만,  
버전이 오래되었거나 시스템용인 경우가 많아 **학습 및 개발용으로는 최신 버전을 별도로 설치하는 것을 권장**합니다.

### ① 공식 사이트 설치
- [Python 공식 다운로드](https://www.python.org/downloads/) 접속
- macOS용 최신 `.pkg` 파일 다운로드 후 실행
- 기본 옵션 그대로 설치

> **📝 용어 설명: 인스톨러(Installer)**
> `.pkg`나 `.exe` 처럼 더블 클릭하면 자동으로 설치를 도와주는 프로그램입니다. 복잡한 설정 없이 "다음 > 다음"만 누르면 되도록 만들어져 있습니다.

**설치 확인:**
```bash
python3 --version
```

**정상 출력 예시:**
```text
Python 3.12.x
```

### ② Homebrew로 설치
Homebrew가 설치되어 있다면 관리 측면에서 가장 깔끔합니다.
```bash
brew update
brew install python
```

> **📝 용어 설명: Homebrew(홈브류)**
> **"개발자를 위한 앱스토어"**라고 생각하면 됩니다. 터미널에서 명령 한 줄로 필요한 프로그램(파이썬, 깃 등)을 설치하고 업데이트할 수 있게 해주는 맥(macOS) 전용 도구입니다.

**설치 확인:**
```bash
python3 --version
```

## 2. Windows에서 파이썬 설치

### ① 공식 사이트 설치 (권장)

* [Python 공식 다운로드](https://www.python.org/downloads/) 접속
* Windows Installer (64-bit) 다운로드 후 실행

**중요: 설치 옵션 체크 (PATH 설정)**
설치 첫 화면에서 반드시 **`Add Python to PATH`** 옵션을 체크해야 합니다.
이것을 체크하지 않으면 터미널에서 `python` 명령을 인식하지 못합니다.

> **📝 용어 설명: PATH (환경 변수)**
> 컴퓨터가 프로그램을 찾을 때 참고하는 **"주소록"**입니다.
> * **체크 안 함:** 컴퓨터가 "파이썬이 어디 있는지 모르겠는데?" 하고 실행을 못 합니다.
> * **체크 함:** 주소록(PATH)에 파이썬 위치가 등록되어, 어디서든 `python`이라고만 치면 실행됩니다.
> 
> 

**설치 확인:**
```bash
python --version
# 또는
py --version
```

**정상 출력 예시:**
```text
Python 3.12.x
```

> **참고: Windows의 `py` 런처**
> `py`는 Windows에 기본 포함되는 **Python Launcher**입니다.
> 여러 파이썬 버전이 설치된 경우 버전 관리에 유리합니다.
> (예: `py -3.12 --version`)

### ② Microsoft Store 설치
* Microsoft Store에서 “Python” 검색 후 설치 가능
* 간편하지만, 향후 실무나 학습 과정에서 경로 문제로 인해 **공식 인스톨러 설치를 더 선호**합니다.

## 3. 파이썬 실행 방식 개요
파이썬은 크게 두 가지 방식으로 실행할 수 있습니다. 상황에 따라 골라 씁니다.

1. **REPL (대화형 실행):** 한 줄씩 실행하며 즉시 결과를 확인 (테스트, 학습용)
2. **스크립트 실행 (`.py` 파일):** 파일 단위로 저장하여 실행 (실제 개발용)

> **📝 용어 설명: REPL vs 스크립트**
> * **REPL(레플):** **"채팅"**과 같습니다. 내가 한 마디 하면(코드 입력), 파이썬이 바로 대답(결과 출력)합니다. 간단한 계산이나 기능 확인용입니다. 부족한 설명을 조금 더 추가하자면  Read-Eval-Print Loop의 약자로, 사용자가 입력한 코드를 즉시 읽고(Read), 평가(Eval)하며, 그 결과를 출력(Print)하고 다시 입력을 받는(Loop) 대화형 프로그래밍 환경 또는 도구입니다.
> 
> * **스크립트(Script):** **"대본"**과 같습니다. 할 말을 파일(`hello.py`)에 미리 다 적어두고, "큐!"(실행) 사인을 주면 처음부터 끝까지 쭉 연기(실행)하는 방식입니다.
> 
> 

---

## 4. REPL 실행하기

터미널에서 아래 명령어를 입력합니다.

* **macOS:** `python3`
* **Windows:** `python` 또는 `py`

정상 실행 시 `>>>` 프롬프트가 나타납니다.

**실행 예시:**

```python
>>> 1 + 2
3
>>> print("Hello Python")
Hello Python
```

**REPL 종료 방법:**

* `exit()` 입력 후 Enter
* **macOS / Linux:** `Ctrl + D`
* **Windows:** `Ctrl + Z` 누른 후 `Enter`

---

## 5. `.py` 파일 실행하기

메모장이나 에디터로 `hello.py` 파일을 생성하고 아래 내용을 적습니다.

**hello.py**

```python
print("Hello World")
print("Python 실행 테스트")
```

**터미널에서 실행:**

```bash
# macOS
python3 hello.py

# Windows
python hello.py
```

**출력 결과:**

```text
Hello World
Python 실행 테스트
```

## 6. VS Code에서 파이썬 실행하기

### ① VS Code 및 확장 설치

* [VS Code 공식 사이트](https://code.visualstudio.com/)에서 설치
* VS Code 실행 후 **Extensions** 탭에서 **Python (Microsoft)** 확장 설치

### ② 파이썬 인터프리터 선택 (중요)

VS Code가 어떤 파이썬을 쓸지 지정해줘야 합니다.

1. **Command Palette 열기:**
* macOS: `Cmd + Shift + P`
* Windows: `Ctrl + Shift + P`


2. **`Python: Select Interpreter`** 검색 후 선택
3. 설치된 **Python 3.x** 버전 선택

### ③ 실행 방법

1. **Run 버튼:** 우측 상단 `▷` 버튼 클릭
2. **단축키:** `Ctrl + F5` (디버깅 없이 실행 - 추천)
3. **터미널:** 하단 터미널 창에서 직접 `python hello.py` 입력

## 참고: 가상 환경 (Virtual Environment)

프로젝트마다 라이브러리 버전이 충돌하는 것을 막기 위해 파이썬에서는 **가상 환경(venv)**을 사용합니다.

> **📝 용어 설명: 가상 환경 (Virtual Environment)**
> 프로젝트를 위한 **"독립된 작업실"**입니다.
> * A 프로젝트는 10평짜리 작업실(가상환경 A)에서 만들고,
> * B 프로젝트는 20평짜리 작업실(가상환경 B)에서 만듭니다.
> * 이렇게 방을 나누면 **A에서 쓰던 도구가 B의 도구와 섞이거나 충돌하지 않아** 안전하게 개발할 수 있습니다.
> 
> 

지금 단계에서는 **“나중에 프로젝트를 할 때는 방을 따로 잡아서 하는구나”** 정도만 알고 있으면 충분합니다.
