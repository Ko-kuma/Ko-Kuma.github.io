## 시작하기 전에..

**튜플(Tuple)**은 리스트(List)와 모습과 기능이 거의 비슷하지만, 가장 결정적인 차이점이 있습니다. 바로 **'값을 바꿀 수 없다'**는 점입니다.

* **리스트**: `[`과 `]`를 사용, 값의 생성·삭제·수정 가능 (Mutable)
* **튜플**: `(`와 `)`를 사용, **값의 삭제·수정 불가능** (Immutable)

## 1. 튜플은 어떻게 만들까?

튜플은 소괄호 `()`를 사용하여 만듭니다.

```python
# 1. 빈 튜플
t1 = ()

# 2. 일반적인 튜플
t2 = (1, 2, 3)

# 3. 괄호 생략 가능 (Pack)
t3 = 1, 2, 3  # (1, 2, 3)과 동일

# 4. 다양한 자료형 포함 및 중첩
t4 = ('a', 'b', ('ab', 'cd'))

```

### [중요] 요소가 1개일 때 주의사항

튜플에 요소를 딱 1개만 저장하고 싶을 때는 반드시 **요소 뒤에 콤마(`,`)**를 붙여야 합니다.

```python
t5 = (1)   # 튜플이 아니라 그냥 정수(int) 1로 인식
t6 = (1,)  # 이것이 요소가 1개인 튜플

print(type(t5)) # <class 'int'>
print(type(t6)) # <class 'tuple'>

```

## 2. 튜플의 요솟값을 지우거나 변경하려고 하면?

튜플은 **불변(Immutable)** 자료형이기 때문에, 리스트처럼 값을 지우거나 바꾸려 하면 **오류가 발생**합니다. 이 특징 때문에 프로그램이 실행되는 동안 **절대 바뀌지 않아야 할 데이터**를 보관할 때 주로 사용합니다.

### 1) 요솟값 삭제 시도 (del)

```python
t1 = (1, 2, 'a', 'b')
del t1[0] 
# 오류 발생: TypeError: 'tuple' object doesn't support item deletion

```

### 2) 요솟값 변경 시도

```python
t1 = (1, 2, 'a', 'b')
t1[0] = 'c'
# 오류 발생: TypeError: 'tuple' object does not support item assignment

```

## 3. 튜플 다루기 (연산)

값을 바꾸는 것 외에는 리스트와 완전히 동일하게 동작합니다.

### 1) 인덱싱 (Indexing)

특정 위치의 값을 가져옵니다.

```python
t1 = (1, 2, 'a', 'b')
print(t1[0])   # 1
print(t1[3])   # 'b'

```

### 2) 슬라이싱 (Slicing)

특정 범위의 값을 자릅니다.

```python
t1 = (1, 2, 'a', 'b')
print(t1[1:])  # (2, 'a', 'b') -> 1번 인덱스부터 끝까지

```

### 3) 튜플 더하기 (+)

두 개의 튜플을 연결하여 **새로운 튜플**을 만듭니다. (기존 튜플이 바뀌는 것이 아님)

```python
t1 = (1, 2)
t2 = (3, 4)
print(t1 + t2) # (1, 2, 3, 4)

```

### 4) 튜플 곱하기 (*)

튜플을 반복하여 **새로운 튜플**을 만듭니다.

```python
t2 = (3, 4)
print(t2 * 3)  # (3, 4, 3, 4, 3, 4)

```

### 5) 튜플 길이 구하기 (len)

```python
t1 = (1, 2, 'a', 'b')
print(len(t1)) # 4

```
