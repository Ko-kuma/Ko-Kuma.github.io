# 1일차 : 파이썬(Python)이란?

## 1. 파이썬의 정의
**파이썬(Python)**은 1991년 네덜란드의 프로그래머 **귀도 반 로섬(Guido van Rossum)**이 발표한  
**고수준(High-level) 프로그래밍 언어**입니다.

쉽게 말해,  
**“사람이 생각하는 방식을 그대로 코드로 옮길 수 있도록 설계된 언어”**라고 볼 수 있습니다.  
컴퓨터의 복잡한 구조를 몰라도 직관적으로 명령을 내릴 수 있는 것이 특징입니다.

또한 파이썬은  
- **동적 타이핑(Dynamic Typing)**을 사용하고  
- **가비지 컬렉션(Garbage Collection)**을 통해 메모리를 자동으로 관리하며  
- 다양한 운영체제에서 동작하는 **범용 프로그래밍 언어**입니다.

파이썬이라는 이름은 뱀에서 온 것이 아니라,  
귀도 반 로섬이 즐겨 보던 영국 BBC의 코미디 프로그램  
**『몬티 파이썬의 날아다니는 서커스(Monty Python’s Flying Circus)』**에서 유래했습니다.

> **핵심 철학 (The Zen of Python)** >  
> “Simple is better than complex.”  
> (단순한 것이 복잡한 것보다 낫다.)
>
> ※ 파이썬에서 `import this`를 실행하면 Zen of Python 전체 문장을 확인할 수 있습니다.

---

### 📝 용어 정리: 언어의 레벨

* **동적 타이핑 (Dynamic Typing)** 변수를 만들 때 자료형(숫자, 문자 등)을 미리 정해두지 않아도, 프로그램이 실행될 때 컴퓨터가 알아서 판단하는 편리한 방식입니다. (예: `a = 10`이라고만 써도 숫자로 인식함)

* **가비지 컬렉션 (Garbage Collection)** 프로그램이 더 이상 사용하지 않는 메모리를 자동으로 찾아서 청소(해제)해주는 기능입니다. 덕분에 개발자가 복잡한 메모리 관리를 신경 쓰지 않아도 됩니다.

* **범용 프로그래밍 언어** 특정 분야(예: 데이터베이스 관리 등)에만 국한되지 않고 웹 개발, 데이터 분석, 인공지능, 게임 등 다양한 분야에서 두루두루 사용할 수 있는 언어를 말합니다.

| 구분                         | 설명                                                                                | 대표 언어                |
| :--------------------------- | :---------------------------------------------------------------------------------- | :----------------------- |
| **고수준 언어 (High-level)** | 사람이 이해하기 쉬운 문법을 사용하며, 컴퓨터 구조를 몰라도 프로그래밍이 가능합니다. | Python, Java, JavaScript |
| **저수준 언어 (Low-level)**  | 컴퓨터 하드웨어(메모리 등)를 직접 제어하며, 성능과 제어력이 뛰어납니다.             | C, Assembly              |

※ **참고: C 언어의 분류** 언어의 레벨 분류는 **국제 표준 기구(ISO)에서 공식적으로 정의한 분류가 아니며**, 교육·실무 맥락에 따라 다르게 사용됩니다.  

C 언어의 경우  
- 공식 표준 문서(C Standard)에서 **‘미들 레벨(Middle-level)’이라는 명칭을 사용하지는 않지만**,  
- 고수준 문법을 가지면서도 저수준 기능을 수행할 수 있다는 특징 때문에 전공 서적이나 개발자들 사이에서 **관용적으로 ‘미들 레벨 언어’라고 불리기도 합니다.**

다만 **파이썬과 같은 고수준 언어와 비교하는 입문 맥락**에서는 C 언어를 **저수준(또는 시스템 언어)**로 분류해도 전혀 틀린 설명이 아닙니다.

## 2. 파이썬의 3가지 핵심 특징

파이썬이 전 세계적으로 널리 사용되는 이유는 다음 세 가지로 정리할 수 있습니다.

### ① 높은 가독성 (직관적인 문법)
- 문법이 간결하여 초보자도 비교적 쉽게 배울 수 있습니다.
- 영어 문장과 비슷한 구조로 되어 있어 코드를 읽기만 해도 의미를 파악하기 쉽습니다.
- **들여쓰기(Indentation)**를 문법으로 강제하여 코드가 자연스럽게 정리됩니다.

### ② 인터프리터 언어 (Interpreter)
- 컴파일 과정을 거치지 않고 코드를 실행합니다.
- 수정 후 바로 결과를 확인할 수 있어 학습과 디버깅에 유리합니다.
- ※ (CPython 기준) 소스 코드는 내부적으로 **바이트코드로 변환된 뒤** 파이썬 가상 머신(Python Virtual Machine)에서 실행됩니다.

### ③ 강력한 생태계 (라이브러리)
- **“Batteries Included”**라는 표현처럼 기본 기능이 매우 풍부합니다.
- 파이썬은 강력한 **표준 라이브러리**를 기본 제공하며, 추가 기능은 **PyPI(파이썬 패키지 저장소)**를 통해 외부 라이브러리로 확장할 수 있습니다.

---

### 📝 용어 정리: 핵심 개념
- **인터프리터 언어:** 코드를 실행 시점에 해석하여 바로 실행하는 언어
- **가독성:** 사람이 코드를 읽고 이해하기 쉬운 정도
- **라이브러리:** 자주 사용하는 기능을 미리 만들어 둔 코드 묶음
- **코드:** 컴퓨터에게 내리는 명령을 프로그래밍 언어로 작성한 문장

## 3. 파이썬으로 할 수 있는 것들 (활용 분야)
파이썬은 **범용 언어**로 거의 모든 분야에서 사용됩니다.

1. **웹 개발**
   - 대규모 웹 서비스의 백엔드
   - Django, Flask

2. **데이터 분석 & AI**
   - 인공지능, 머신러닝, 데이터 분석
   - Pandas, PyTorch, TensorFlow

3. **업무 자동화**
   - 엑셀 자동 처리, 이메일 발송, 웹 크롤링
   - 파이썬 스크립트를 이용한 반복 업무 자동화

4. **GUI 프로그래밍**
   - 데스크톱 프로그램 제작
   - PyQt, Tkinter

## 4. 코드 비교 (Java vs Python)

**“Hello World”**를 출력하는 가장 기본적인 예제입니다.

### Java
```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```

### Python
직관적으로 **'출력하라(print)'**는 명령만 내리면 바로 실행됩니다.
```python
print("Hello World")
```
Java에 비해 코드가 훨씬 간결하며,
이러한 점이 파이썬이 입문 언어로 많이 선택되는 이유 중 하나입니다.
