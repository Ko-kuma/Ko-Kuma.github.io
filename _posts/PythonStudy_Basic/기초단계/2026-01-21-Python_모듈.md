# 시작하기 전에..
비슷한 기능을 가진 함수나 클래스들을 별도의 파일로 나누어 관리하는, 마치 공구함처럼 필요한 도구(함수, 클래스)를 모아놓은 파일, **모듈**에 대해 정리합니다.



## 1. 모듈이란?

모듈이란 함수나 변수, 클래스들을 모아놓은 **파이썬 파일(.py)** 그 자체를 의미합니다.
* 우리가 만든 파이썬 파일 하나하나가 모두 모듈이 될 수 있습니다.
* 다른 사람이 만든 모듈을 불러와 사용할 수도 있습니다. (라이브러리)

## 2. 모듈 만들고 불러오기

### 모듈 만들기 (`mod1.py`)
간단한 더하기, 빼기 기능이 있는 모듈을 만들어 봅시다. 메모장이나 에디터로 아래 코드를 작성하고 `mod1.py`로 저장합니다.

```python
# mod1.py
def add(a, b):
    return a + b

def sub(a, b):
    return a - b

```

### 모듈 불러오기 (`import`)

이제 같은 폴더(디렉터리)에서 IDLE이나 터미널을 열고 모듈을 사용해 봅니다.

```python
import mod1

print(mod1.add(3, 4)) # 7
print(mod1.sub(4, 2)) # 2

```

* **사용법:** `import 모듈이름` (확장자 .py는 뺍니다)
* **호출:** `모듈이름.함수이름()`

## 3. `if __name__ == "__main__":` 이란?

모듈을 공부하다 보면 파일 맨 아래에 이런 코드를 자주 보게 됩니다.

```python
if __name__ == "__main__":
    print(add(3, 4))
    print(sub(4, 2))

```

### `__name__` 변수란?

파이썬이 내부적으로 사용하는 특별한 변수입니다.

1. **직접 실행할 때:** 파일을 직접 실행(`python mod1.py`)하면 `__name__` 변수에는 **`"__main__"`**이라는 값이 들어갑니다.
2. **불러와서 사용할 때:** 다른 파일에서 `import mod1`을 하면 `__name__` 변수에는 **모듈 이름(`"mod1"`)**이 들어갑니다.

### 의미와 사용하는 이유

만약 위 코드가 없다면, `import mod1`을 하는 순간 `print(add(3, 4))`가 자동으로 실행되어 버립니다.
우리는 함수만 빌려 쓰고 싶은데, 모듈 안의 테스트 코드까지 실행되면 안 되겠죠?

> **"이 파일이 메인 프로그램으로 직접 실행될 때만 아래 코드를 실행해라!"** 라는 뜻입니다.

## 4. 클래스나 변수 등을 포함한 모듈

모듈 안에는 함수뿐만 아니라 변수나 클래스도 넣을 수 있습니다.

```python
# mod2.py
PI = 3.141592  # 변수

class Math:    # 클래스
    def solv(self, r):
        return PI * (r ** 2)

def add(a, b): # 함수
    return a + b

```

**사용하기:**

```python
import mod2

print(mod2.PI)      # 변수 사용
a = mod2.Math()     # 클래스 사용
print(a.solv(2)) 

```

## 5. 다른 파일에서 모듈 불러오기 (Import 변형)

`mod1.add` 처럼 매번 모듈 이름을 붙이기 귀찮다면 `from`을 사용합니다.

```python
# 1. add 함수 하나만 불러오기
from mod1 import add
print(add(3, 4)) # mod1. 없이 바로 사용 가능

# 2. 모든 함수 불러오기
from mod1 import *
print(sub(4, 2))

```

## 6. 다른 디렉터리에 있는 모듈을 불러오는 방법

지금까지는 실행하는 파일과 모듈이 **같은 폴더**에 있을 때였습니다.
만약 모듈이 다른 폴더(`C:\doit\mymod`)에 있다면 어떻게 해야 할까요?

### 방법 ① `sys.path.append` 사용하기

파이썬 라이브러리인 `sys` 모듈을 이용하면 파이썬이 모듈을 검색하는 경로(`sys.path`)에 내 폴더를 추가할 수 있습니다.

```python
import sys

# 파이썬에게 모듈 검색 경로를 추가해줌
sys.path.append("C:/doit/mymod")

import mod2 # 이제 불러올 수 있음
print(mod2.add(3, 4))

```

### 방법 ② `PYTHONPATH` 환경 변수 사용하기

코드를 수정하지 않고, 운영체제(OS) 환경 변수를 설정하는 방법입니다.

**터미널(CMD)에서 실행:**

```bash
C:\doit> set PYTHONPATH=C:\doit\mymod
C:\doit> python
>>> import mod2
>>> print(mod2.add(3, 4))

```

* **리눅스/맥:** `export PYTHONPATH=/home/user/mymod`

---

**요약:**

1. **모듈**은 `.py` 파일 그 자체다.
2. **`if __name__ == "__main__":`**은 import 당할 때는 실행되지 않게 막아주는 안전장치다.
3. 다른 폴더의 모듈은 **`sys.path.append()`**를 이용해 경로를 추가한 뒤 불러온다.
