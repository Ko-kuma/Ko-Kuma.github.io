## 1. 홀수, 짝수 판별하기

### 문제 1 (함수 기초)
주어진 숫자가 홀수인지 짝수인지 판별하여, 짝수면 "Even", 홀수면 "Odd"를 반환(Return)하는 함수 `is_odd(number)`를 작성하세요.

### 문제 2 (함수와 사용자 입력)
위에서 만든 `is_odd` 함수를 사용하여, 사용자에게 숫자를 입력받고 그 결과를 출력하는 프로그램을 작성하세요.

<br>

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이 1**
```python
def is_odd(number):
    if number % 2 == 0:
        return "Even"
    else:
        return "Odd"

```

**풀이 2**
`input()`은 문자열을 반환하므로 `int()`로 변환해야 함에 유의합니다.

```python
# is_odd 함수가 정의되어 있다고 가정
user_input = input("숫자를 입력하세요: ")
number = int(user_input)
print(is_odd(number))

```

</details>

## 2. 모든 입력의 평균값 구하기

### 문제 3 (가변 매개변수 함수)

입력값이 몇 개가 들어오든 상관없이, 들어온 모든 수의 평균값을 구하는 함수 `get_avg`를 작성하세요. (힌트: `*args` 사용)

### 문제 4 (프로그램 입출력 - sys)

터미널에서 `python avg.py 10 20 30` 처럼 실행했을 때, 입력된 숫자들의 평균을 구해 출력하는 `avg.py` 프로그램을 작성하세요.

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이 3**

```python
def get_avg(*args):
    result = 0
    for i in args:
        result += i
    return result / len(args)

print(get_avg(1, 2, 3, 4, 5)) # 결과: 3.0

```

**풀이 4**
`sys.argv`를 사용합니다. `sys.argv[0]`은 파일명이므로 제외하고, 나머지 요소들을 정수로 변환해야 합니다.

```python
# avg.py 파일 내용
import sys

args = sys.argv[1:] # 파일명을 제외한 나머지 인수
total = 0

for i in args:
    total += int(i) # 입력값은 문자열이므로 int 변환 필수

print(total / len(args))

```

</div>
</details>

## 3. 프로그램 오류 수정하기

### 문제 5 (파일 입출력 모드)

다음은 파일(`test.txt`)에 "Hello Python"이라는 문자열을 쓰려고 작성한 코드입니다. 하지만 오류가 발생합니다. 원인을 찾아 고치세요.

```python
f = open("test.txt", "r")
f.write("Hello Python")
f.close()

```

### 문제 6 (사용자 입력 자료형)

사용자에게 숫자 두 개를 입력받아 더한 값을 출력하려는데, `3`과 `4`를 입력했더니 `7`이 아니라 `34`가 출력됩니다. 올바르게 수정하세요.

```python
a = input("첫 번째 수: ")
b = input("두 번째 수: ")
print(a + b)

```

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이 5**
파일을 **쓰기** 위해서는 모드를 `'w'`(덮어쓰기) 또는 `'a'`(이어쓰기)로 열어야 합니다. `'r'`은 읽기 전용입니다.

```python
f = open("test.txt", "w") # "r" -> "w" 로 변경
f.write("Hello Python")
f.close()

```

**풀이 6**
`input()`은 입력을 무조건 **문자열**로 받습니다. 정수로 변환(`int`)해 주어야 산술 연산이 됩니다.

```python
a = int(input("첫 번째 수: "))
b = int(input("두 번째 수: "))
print(a + b)

```

</div>
</details>

## 5. 사용자 입력 저장하기 (파일 I/O)

### 문제 8 (파일 쓰기)

사용자로부터 입력을 받아 `result.txt` 파일에 저장하는 프로그램을 작성하세요. (단, 파일을 열 때 `with`문을 사용하세요.)

### 문제 9 (파일 내용 추가)

위에서 만든 `result.txt`에, 프로그램을 다시 실행해서 새로운 내용을 입력해도 기존 내용이 지워지지 않고 **뒤에 덧붙여지도록** 코드를 작성하세요.

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이 8**
쓰기 모드(`'w'`)를 사용합니다.

```python
user_input = input("저장할 내용을 입력하세요: ")
with open("result.txt", "w") as f:
    f.write(user_input)

```

**풀이 9**
추가 모드(`'a'`)를 사용합니다. 줄바꿈을 위해 `\n`을 추가해주면 더 좋습니다.

```python
user_input = input("추가할 내용을 입력하세요: ")
with open("result.txt", "a") as f:
    f.write(user_input + "\n")

```

</div>
</details>
