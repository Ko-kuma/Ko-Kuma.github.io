# 시작하기 전에..
프로그래밍이 단순히 위에서 아래로 읽어 내려가는 글이 아니라, 상황에 따라 다르게 동작하도록 만드는 핵심, **if문(조건문)**에 대해 정리해 봅니다.

## 1. if문은 왜 필요할까?

우리의 일상생활은 선택의 연속입니다.
> "비가 오면 우산을 쓰고, 비가 안 오면 그냥 간다."

프로그래밍도 마찬가지입니다. 모든 코드를 순서대로만 실행하는 것이 아니라, **특정 조건에 따라 다른 코드를 실행**해야 할 때가 반드시 생깁니다. 이때 사용하는 것이 바로 `if`문입니다.

## 2. IDLE 셸(Shell) 사용 시 주의사항

파이썬 IDLE의 대화형 셸(Shell)에서 `if`문을 연습할 때는 다음을 주의해야 합니다.

1.  **들여쓰기(Indentation):** 셸에서는 자동으로 들여쓰기가 되지 않을 수 있으므로, `if`문 다음 줄에서 직접 스페이스바(4번)나 탭을 눌러 들여쓰기를 해야 합니다.
2.  **입력 완료:** `if`문 작성이 끝났다면 **엔터 키를 두 번** 눌러야 명령이 실행됩니다.

## 3. if문의 기본 구조와 콜론(:)

파이썬의 if문은 구조가 매우 직관적입니다.

```python
if 조건문:
    수행할_문장1
    수행할_문장2
else:
    수행할_문장A
    수행할_문장B

```

**주의할 점 :** 조건문 뒤에는 반드시 **콜론(:)**을 붙여야 합니다. 이는 "이제부터 조건에 해당할 때 실행할 내용이 나온다"는 신호입니다.

## 4. 들여쓰기(Indentation) 방법

다른 언어(C, Java 등)는 중괄호 `{}`로 블록을 구분하지만, **파이썬은 들여쓰기로 블록을 구분**합니다.

* **규칙:** `if`문에 속하는 모든 문장은 **반드시 같은 너비**로 들여쓰기를 해야 합니다.
* **권장:** 공백(Space) **4칸**을 사용하는 것이 파이썬의 표준 권장사항입니다. (탭(Tab) 1번을 써도 되지만, 공백과 탭을 섞어 쓰면 오류가 발생합니다.)

## 5. 조건문이란 무엇인가?

조건문이란 **참(True)**과 **거짓(False)**을 판단할 수 있는 문장을 말합니다. 주로 비교 연산자와 논리 연산자를 사용합니다.

### ① 비교 연산자

값을 비교할 때 사용합니다.

| 연산자   | 설명                  | 예시 (a=3, b=5) | 결과  |
| -------- | --------------------- | --------------- | ----- |
| `x < y`  | x가 y보다 작다        | `a < b`         | True  |
| `x > y`  | x가 y보다 크다        | `a > b`         | False |
| `x == y` | x와 y가 같다          | `a == b`        | False |
| `x != y` | x와 y가 같지 않다     | `a != b`        | True  |
| `x >= y` | x가 y보다 크거나 같다 | `a >= b`        | False |
| `x <= y` | x가 y보다 작거나 같다 | `a <= b`        | True  |

> **주의:** 같다는 뜻은 `=`가 아니라 `==`입니다. (`=`는 변수에 값을 대입할 때 씁니다.)

### ② 논리 연산자 (and, or, not)

여러 조건을 조합할 때 사용합니다.

| 연산자    | 설명                               | 예시                        | 비유        |
| --------- | ---------------------------------- | --------------------------- | ----------- |
| `x or y`  | x와 y 둘 중 **하나만 참이어도** 참 | `돈이 있거나 카드가 있다면` | 택시를 탄다 |
| `x and y` | x와 y **둘 다 참이어야** 참        | `돈도 있고 시간도 있다면`   | 영화를 본다 |
| `not x`   | x가 **거짓이면** 참 (반대)         | `돈이 없다면`               | 걸어간다    |

### ③ in, not in (멤버십 연산자)

리스트, 튜플, 문자열 안에 특정 값이 있는지 확인할 때 사용합니다. 파이썬만의 아주 편리한 기능입니다.

| 연산자            | 설명                        | 예시 (`pocket = ['paper', 'money', 'cellphone']`) | 결과 |
| ----------------- | --------------------------- | ------------------------------------------------- | ---- |
| `x in 리스트`     | 리스트 안에 x가 **있는가?** | `'money' in pocket`                               | True |
| `x not in 리스트` | 리스트 안에 x가 **없는가?** | `'card' not in pocket`                            | True |

## 6. 조건문에서 아무 일도 하지 않게 설정하려면?

조건을 만들었지만, 딱히 실행할 코드가 없거나 나중에 코드를 채워 넣고 싶을 때 `pass`를 사용합니다.

```python
pocket = ['paper', 'money']
if 'money' in pocket:
    pass  # 돈이 있으면 가만히 있는다 (아무 일도 안 함)
else:
    print("카드를 꺼내라")

```

## 7. 다양한 조건을 판단하는 elif

조건이 두 개(참/거짓)가 아니라 여러 개일 때는 `elif` (else if의 줄임말)를 사용합니다.

```python
pocket = ['paper', 'cellphone']
card = True

if 'money' in pocket:
    print("택시를 타고 가라")
elif card: 
    print("택시를 타고 가라") # 돈은 없지만 카드가 있는 경우
else:
    print("걸어가라")

```

`if`, `elif`, `else` 구조를 가지며, `elif`는 개수에 제한 없이 사용할 수 있습니다.

## 8. if문 한 줄로 작성하기

수행할 문장이 한 줄뿐이라면 줄 바꿈 없이 간략하게 작성할 수 있습니다.

```python
pocket = ['paper', 'money']
if 'money' in pocket: pass
else: print("카드를 꺼내라")

```

## 9. 조건부 표현식 (Conditional Expression)

변수에 값을 대입할 때, 조건에 따라 다른 값을 넣고 싶다면 **조건부 표현식**을 사용하여 한 줄로 깔끔하게 쓸 수 있습니다. 다른 언어의 '삼항 연산자'와 비슷합니다.

* **구조:** `참일_때_값 if 조건문 else 거짓일_때_값`

```python
score = 60

# 일반적인 if-else 문
if score >= 60:
    message = "success"
else:
    message = "failure"

# 조건부 표현식 사용 
message = "success" if score >= 60 else "failure"

```
