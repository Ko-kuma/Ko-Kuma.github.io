## 1. 조건문의 참과 거짓 판별

### 문제
다음 코드의 실행 결과는 무엇일까요?
```python
a = "Life is too short, you need python"

if "wife" in a: print("wife")
elif "python" in a and "you" not in a: print("python")
elif "shirt" not in a: print("shirt")
elif "need" in a: print("need")
else: print("none")

```

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**정답: shirt**

**풀이:**

1. `if "wife" in a`: 거짓(False) → 넘어감
2. `elif "python" in a and "you" not in a`: "python"은 있지만 "you"도 있으므로 거짓(False) → 넘어감
3. `elif "shirt" not in a`: "shirt"라는 단어는 문장에 없으므로 **참(True)** → "shirt" 출력 후 if문 종료
4. (참고) `elif "need" in a`는 참이지만, 앞선 3번 조건이 먼저 만족되었으므로 실행되지 않습니다.

</div>
</details>

## 2. 3의 배수의 합 구하기

### 문제

1부터 1000까지의 자연수 중 3의 배수의 합을 `while`문을 사용하여 구하세요.

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이**

```python
result = 0
i = 1
while i <= 1000:
    if i % 3 == 0:  # 3으로 나누어 떨어지면(3의 배수면)
        result += i
    i += 1

print(result)
# 결과: 166833

```

</div>
</details>

## 3. 2의 배수의 합 구하기

### 문제

1부터 100까지의 자연수 중 2의 배수(짝수)의 합을 `for`문과 `range`를 사용하여 구하세요.

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이**

```python
result = 0
for i in range(1, 101): # 1부터 100까지
    if i % 2 == 0:      # 2의 배수라면
        result += i

print(result)
# 결과: 2550

```

</div>
</details>

## 4. 별(*) 표시하기

### 문제

`while`문 또는 `for`문을 사용하여 다음과 같이 별(`*`)을 표시하는 프로그램을 작성하세요.

```
*
**
***
****
*****

```

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이 1 (while문 사용)**

```python
i = 0
while True:
    i += 1
    if i > 5: break     # i가 5를 넘으면 종료
    print('*' * i)      # 문자열에 숫자를 곱하면 그만큼 반복됨

```

**풀이 2 (for문 사용 - 추천)**

```python
for i in range(1, 6):   # 1부터 5까지 반복
    print('*' * i)

```

</div>
</details>

## 6. 학급의 평균 점수 구하기

### 문제

A 학급에 총 10명의 학생이 있고, 이들의 중간고사 점수는 다음과 같습니다.
`[70, 60, 55, 75, 95, 90, 80, 80, 85, 100]`
`for`문을 사용하여 A 학급의 평균 점수를 구하세요.

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이**

```python
A = [70, 60, 55, 75, 95, 90, 80, 80, 85, 100]
total = 0

for score in A:
    total += score  # 점수를 모두 더함

average = total / len(A) # 총점 / 학생수
print(average)
# 결과: 79.0

```

</div>
</details>

## 7. 리스트 컴프리헨션 사용하기

### 문제

다음 리스트 `numbers`에서 **홀수에만 2를 곱하여** 저장하는 새로운 리스트 `result`를 만드세요.
(단, **리스트 컴프리헨션**을 사용해야 합니다.)

`numbers = [1, 2, 3, 4, 5]`

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이**
구조: `[표현식 for 항목 in 리스트 if 조건]`

```python
numbers = [1, 2, 3, 4, 5]
result = [n * 2 for n in numbers if n % 2 == 1]
print(result)
# 결과: [2, 6, 10]

```

* **해설:**
1. `numbers`에서 `n`을 하나씩 꺼낸다.
2. `if n % 2 == 1` (홀수인지 확인한다).
3. 참이면 `n * 2`를 계산하여 리스트에 담는다.



</div>
</details>
