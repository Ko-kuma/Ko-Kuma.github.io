# 변수에 대해
## 1. 변수(Variable)란?
변수는 **데이터(값)에 붙이는 이름표**입니다.

복잡한 숫자나 긴 문장을 매번 머릿속으로 기억하기 힘들기 때문에, 컴퓨터에게 "이 값은 앞으로 '내점수'라고 부를게!"라고 약속하는 것과 같습니다.

> **비유**
> * **마트에서 산 물건 (데이터)**
> * **물건을 담은 상자에 붙인 라벨 (변수)**
<br>

## 2. 변수 만들기 (선언과 할당)

파이썬에서 변수를 만드는 방법은 아주 간단합니다. **등호(`=`)** 기호를 사용합니다.

```python
# 변수이름 = 저장할값
age = 25
name = "철수"

```

* **변수명 (왼쪽):** 내가 지어준 이름 (`age`, `name`)
* **값 (오른쪽):** 실제 데이터 (`25`, `"철수"`)
* **`=` (등호):** 수학의 "같다"가 아닙니다! **"오른쪽의 값을 왼쪽 이름표(변수)에 넣어라"** 라는 뜻입니다.

## 3. 변수 이름 짓는 규칙 (Naming Rules)

변수 이름은 마음대로 지을 수 있지만, 파이썬 에서의 문법을 지켜야 에러가 나지 않습니다.

### 반드시 지켜야 할 규칙 (필수)

1. **영문자, 숫자, 언더바(`_`)**만 사용할 수 있습니다.
2. **숫자로 시작할 수 없습니다.**
* `1grade` (에러)
* `grade1` (가능)


3. **띄어쓰기(공백)를 할 수 없습니다.**
* `my score` (에러)
* `my_score` (가능)


4. **대소문자를 엄격하게 구별합니다.**
* `apple` 과 `Apple`은 서로 다른 변수입니다.



### 개발자들의 약속 (관례)

* **스네이크 케이스 (Snake Case):** 단어와 단어 사이를 언더바(`_`)로 연결합니다. 뱀처럼 보인다고 해서 이렇게 부릅니다.
* `student_name` (파이썬에서 선호하는 방식)
* `studentName` (자바 등 다른 언어 스타일)



## 4. 파이썬 변수의 특징

### ① 자료형을 미리 정하지 않아도 됩니다.

C언어나 자바 같은 다른 언어는 변수를 만들 때 "이건 숫자야!", "이건 글자야!"라고 미리 신고해야 합니다. 하지만 파이썬은 똑똑해서 **값을 넣으면 알아서 타입을 인식**합니다.

```python
a = 10       # 알아서 정수로 인식
b = "Hello"  # 알아서 문자로 인식

```

### ② 변수의 값은 언제든 바꿀 수 있습니다.

'변수(變數)'라는 한자 뜻 그대로 **'변할 수 있는 수'**입니다.

```python
score = 100
print(score)  # 100 출력

score = 50    # 값을 50으로 변경 (재할당)
print(score)  # 50 출력 (기존 100은 사라짐)

```

### ③ 변수끼리 계산할 수 있습니다.

변수에 저장된 값을 이용해 더하기, 빼기 등 다양한 연산이 가능합니다.

```python
apple = 1000
count = 3
total = apple * count  # 1000 * 3

print(total)  # 3000 출력

```

## 📝 정리

1. 변수는 데이터에 붙이는 **이름표**다.
2. `변수명 = 값` 형태로 만든다. (오른쪽 값을 왼쪽에 할당)
3. 이름 지을 때 **숫자로 시작 금지**, **띄어쓰기 금지**.
4. 변수의 값은 언제든지 **변경(재할당)**할 수 있다.
