## 1. 변수(Variable)란?

변수는 **데이터(값)를 가리키는 이름표**입니다.

복잡한 숫자나 긴 문장을 매번 머릿속으로 기억하기 힘들기 때문에, 컴퓨터 메모리 어딘가에 있는 값에 "이 값은 앞으로 '내점수'라고 부를게!"라고 이름표를 붙여두는 것과 같습니다.

> **비유**
> * **마트에서 산 물건 (데이터)**
> * **물건을 담은 상자에 붙인 라벨 (변수)**

---

## 2. 변수 만들기 (선언과 할당)

파이썬에서 변수를 만드는 방법은 **등호(`=`)** 기호를 사용합니다.

```python
# 변수이름 = 저장할값
age = 25
name = "철수"

```

* **변수명 (왼쪽):** 내가 지어준 이름 (`age`, `name`)
* **값 (오른쪽):** 실제 데이터 (`25`, `"철수"`)
* **`=` (등호):** 수학의 "같다"가 아닙니다! **"오른쪽의 값을 왼쪽 이름표(변수)가 가리키게 하라"**는 할당(Assignment)의 의미입니다.

---

## 3. 변수를 만드는 여러 가지 방법

파이썬은 다른 언어보다 훨씬 유연한 변수 할당 방식을 지원합니다.

### 1) 튜플 언패킹 (Tuple Unpacking)

여러 개의 변수에 여러 개의 값을 한 번에 넣을 수 있습니다.

```python
a, b = 'python', 'life'
# a에는 'python', b에는 'life'가 들어갑니다.
print(a) # python

```

### 2) 튜플로 묶어서 할당

괄호를 써도 되고 안 써도 됩니다.

```python
(a, b) = 'python', 'life'
[c, d] = ['python', 'life'] # 리스트도 가능

```

### 3) 여러 변수에 같은 값 할당

```python
a = b = 'python'
# a도 'python', b도 'python'을 가리킵니다.

```

### 4) 두 변수의 값 바꾸기 (Swapping) ⭐

다른 언어에서는 임시 변수(`temp`)가 필요하지만, 파이썬은 아주 간단합니다.

```python
a = 3
b = 5
a, b = b, a  # 서로 값을 맞바꿈
print(a) # 5
print(b) # 3

```

## 4. [중요] 리스트 복사하기 (Memory Reference)

변수를 다룰 때 가장 주의해야 할 점은 **객체의 주소(Reference)** 개념입니다. 특히 리스트를 복사할 때 실수를 많이 합니다.

### 1) 잘못된 복사 (`=`)

```python
a = [1, 2, 3]
b = a          # b는 a와 '같은 주소'를 가리킴 (이름표만 하나 더 붙인 셈)

a[1] = 4       # a를 수정했는데
print(b)       # [1, 4, 3] -> b도 같이 바뀜!

```

우리가 원하는 것은 `a`와 똑같은 값을 가진 **별개의 리스트 `b`**를 만드는 것입니다. 이를 위해 다음 3가지 방법을 사용합니다.

### 2) 슬라이싱 이용하기 (`[:]`)

리스트 전체 범위(`[:]`)를 잘라내면 새로운 리스트가 생성됩니다.

```python
a = [1, 2, 3]
b = a[:]       # 값을 복사해서 새로운 리스트 생성

a[1] = 4
print(a)       # [1, 4, 3]
print(b)       # [1, 2, 3] (b는 영향받지 않음!)

```

### 3) `copy` 모듈 이용하기

파이썬 내장 모듈인 `copy`를 import 해서 사용합니다.

```python
from copy import copy
a = [1, 2, 3]
b = copy(a)    # b = a[:] 와 동일

```

### 4) 리스트 내장 함수 `copy()` 이용하기

리스트 자료형이 자체적으로 가지고 있는 함수입니다. (가장 직관적이고 추천하는 방법)

```python
a = [1, 2, 3]
b = a.copy()

```

## 5. 변수 이름 짓는 규칙 (Naming Rules)

변수 이름은 마음대로 지을 수 있지만, 파이썬 문법을 지켜야 합니다.

1. **영문자, 숫자, 언더바(`_`)**만 사용 가능.
2. **숫자로 시작 금지** (`1grade` X, `grade1` O).
3. **띄어쓰기 금지** (`my score` X, `my_score` O).
4. **예약어 사용 금지** (`if`, `for`, `while`, `True` 등).

> **관례 (Convention)**
> 파이썬 개발자들은 주로 **스네이크 케이스(`snake_case`)**를 사용합니다. (단어 사이를 `_`로 연결)
