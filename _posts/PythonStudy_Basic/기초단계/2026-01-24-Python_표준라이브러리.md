## 1. 개요: 표준 라이브러리 vs 내장 함수

가장 먼저 이 둘의 차이를 명확히 알고 넘어가야 합니다.

|     구분      | **내장 함수 (Built-in)**               | **표준 라이브러리 (Standard Library)**              |
| :-----------: | :------------------------------------- | :-------------------------------------------------- |
|   **정의**    | 파이썬 인터프리터에 이미 녹아있는 함수 | 별도의 모듈 파일로 존재하며 가져다 써야 하는 도구들 |
|  **준비물**   | 없음 (설치 불필요)                     | **`import 모듈명`** 구문 필요 (설치 불필요)         |
| **대표 예시** | `print`, `len`, `input`, `int`         | `math`, `os`, `sys`, `datetime`, `json`             |

## 2. 날짜와 시간 (`datetime`, `time`)

### 2-1. `datetime.date`
> **정의:** '년-월-일' 정보를 다루는 객체입니다. 날짜 간의 뺄셈(D-day 계산)이 가능합니다.

```python
import datetime

day1 = datetime.date(2024, 1, 1)
day2 = datetime.date(2024, 4, 10)

diff = day2 - day1
print(diff.days) # 100 (일수 차이)
print(day1.weekday()) # 0: 월요일 ~ 6: 일요일

```

### 2-2. `time` 모듈

> **정의:** 시스템 시간(UTC)을 측정하거나, 프로그램의 실행 흐름을 잠시 멈추는(Sleep) 기능을 제공합니다.

**[주요 함수 정리]**

| 함수명                 | 설명                        | 특징                                      |
| ---------------------- | --------------------------- | ----------------------------------------- |
| **`time.time()`**      | 현재 시간(초) 반환          | 1970.1.1부터 흐른 실수형 시간 (`1705...`) |
| **`time.localtime()`** | 연월일시분초로 변환         | `struct_time` 객체 반환                   |
| **`time.asctime()`**   | 보기 편한 문자열로 변환     | `'Sun Jan 25 ...'` 형태                   |
| **`time.ctime()`**     | `time()` 값을 바로 문자열로 | `asctime`과 결과 동일                     |
| **`time.sleep(n)`**    | `n`초 동안 대기             | 코드 실행을 잠시 멈춤                     |

* **참고:** `localtime`이나 `asctime`은 인수를 넣지 않으면 **현재 시간**을 기준으로 동작합니다.

### 2-3. `time.strftime` (시간 포맷팅)

> **정의:** 날짜/시간 데이터를 원하는 형식의 **문자열**로 변환합니다. (String Format Time)

**[포맷 코드 표]**

| 코드 | 설명        | 예시   | 코드 | 설명          | 예시 |
| ---- | ----------- | ------ | ---- | ------------- | ---- |
| `%Y` | 4자리 연도  | 2026   | `%H` | 24시간제 시간 | 14   |
| `%m` | 월 (01~12)  | 01     | `%I` | 12시간제 시간 | 02   |
| `%d` | 일 (01~31)  | 25     | `%M` | 분            | 30   |
| `%A` | 요일 (전체) | Sunday | `%S` | 초            | 59   |

```python
import time
# 현재 시간을 포맷에 맞춰 출력
print(time.strftime('%Y-%m-%d %H:%M:%S', time.localtime(time.time())))

```

## 3. 수학과 난수 (`math`, `random`)

### 3-1. `math`

> **정의:** 최대공약수, 최소공배수 등 정교한 수학 연산을 지원합니다.

| 함수명               | 설명                                 | 예시                     |
| -------------------- | ------------------------------------ | ------------------------ |
| **`math.gcd(a, b)`** | 최대공약수 (Greatest Common Divisor) | `math.gcd(60, 80)` -> 20 |
| **`math.lcm(a, b)`** | 최소공배수 (Least Common Multiple)   | `math.lcm(15, 25)` -> 75 |

```python
import math
print(math.gcd(60, 100, 80)) # 3개 숫자도 가능

```

### 3-2. `random`

> **정의:** 규칙 없는 무작위 수(난수)를 생성하거나 데이터를 섞습니다.

| 함수명                     | 설명                      | 예시                       |
| -------------------------- | ------------------------- | -------------------------- |
| **`random.random()`**      | 0.0 ~ 1.0 사이 실수 생성  | `0.342...`                 |
| **`random.randint(a, b)`** | a 이상 b 이하 정수 생성   | `random.randint(1, 45)`    |
| **`random.choice(list)`**  | 리스트 중 1개 무작위 선택 | `random.choice(['A','B'])` |
| **`random.shuffle(list)`** | 리스트 순서 섞기          | `random.shuffle(data)`     |

## 4. 데이터 가공과 순회 (`itertools`)

> **정의:** 리스트 같은 반복 가능 객체를 효율적으로 다루기 위한 '반복자 생성' 도구 모음입니다.

### 4-1. `zip_longest`

`zip()`은 길이가 짧은 쪽에 맞추지만, 이것은 긴 쪽에 맞춰 나머지를 `fillvalue`로 채웁니다.

```python
import itertools
students = ['A', 'B']
snacks = ['사탕', '젤리', '초콜릿']
print(list(itertools.zip_longest(students, snacks, fillvalue='없음')))
# 결과: [('A', '사탕'), ('B', '젤리'), ('없음', '초콜릿')]

```

### 4-2. 순열과 조합 (매우 중요!)

| 함수                                | 이름         | 순서 중요? | 중복 허용? | 예시 (2개 뽑기) |
| ----------------------------------- | ------------ | ---------- | ---------- | --------------- |
| **`permutations`**                  | **순열**     | **O**      | X          | (A,B) != (B,A)  |
| **`combinations`**                  | **조합**     | **X**      | X          | (A,B) == (B,A)  |
| **`combinations_with_replacement`** | **중복조합** | X          | **O**      | (A,A) 가능      |

```python
import itertools
data = [1, 2]

# 순열: 순서가 다르면 다른 것
print(list(itertools.permutations(data, 2))) 
# [(1, 2), (2, 1)]

# 조합: 순서 상관없이 구성 요소가 같으면 같은 것
print(list(itertools.combinations(data, 2))) 
# [(1, 2)]

```

## 5. 함수형 프로그래밍 (`functools`, `operator`)

### 5-1. `functools.reduce`

> **정의:** 데이터를 처음부터 끝까지 순회하며 함수를 적용해 **단 하나의 값**으로 만듭니다.

```python
import functools

data = [3, 1, 5, 2, 4]

# 예제: 최댓값 구하기
# x: 현재까지의 승자(큰 값), y: 새로운 도전자
max_val = functools.reduce(lambda x, y: x if x > y else y, data)
print(max_val) # 5

```

### 5-2. `operator` (정렬 기준)

> **정의:** 정렬(`sorted`) 시 무엇을 기준으로 정렬할지 키(Key)를 쉽게 추출해 줍니다.

| 함수명           | 대상                   | 설명                              |
| ---------------- | ---------------------- | --------------------------------- |
| **`itemgetter`** | 리스트, 튜플, 딕셔너리 | 인덱스나 Key를 기준으로 정렬      |
| **`attrgetter`** | 클래스 객체            | 객체의 속성(변수)을 기준으로 정렬 |

```python
from operator import itemgetter

students = [("jane", 22), ("dave", 32)]
# 인덱스 1(나이)을 기준으로 정렬
print(sorted(students, key=itemgetter(1)))

```

## 6. 파일 및 운영체제 제어 (`os`, `shutil`, `glob`)

### 6-1. `os` (운영체제 제어)

> **정의:** 환경 변수, 디렉터리, 시스템 명령어 등 OS의 기능을 파이썬에서 제어합니다.

| 함수             | 설명                                      | 예제                 |
| ---------------- | ----------------------------------------- | -------------------- |
| **`os.environ`** | 내 시스템 환경 변수값 보기 (딕셔너리)     | `os.environ['PATH']` |
| **`os.chdir`**   | 디렉터리 위치 변경 (Change Directory)     | `os.chdir('/tmp')`   |
| **`os.getcwd`**  | 현재 디렉터리 위치 확인 (Get CWD)         | `print(os.getcwd())` |
| **`os.system`**  | 시스템 명령어 실행 (결과값만 반환)        | `os.system("ls")`    |
| **`os.popen`**   | 명령어 실행 후 **출력값**을 파일처럼 읽기 | `f = os.popen("ls")` |

### 6-2. `shutil` (파일 고수준 연산)

> **정의:** 파일 복사, 이동, 삭제 등의 작업을 손쉽게 처리합니다.

**[예제: 파일을 이동시켜 '안전한 삭제' 기능 만들기]**

```python
import shutil
import os

def move_to_trash(filename):
    trash_dir = "./Trash"
    if not os.path.exists(trash_dir):
        os.mkdir(trash_dir)
    # 파일을 이동시킴 (실제 삭제가 아닌 이동)
    shutil.move(filename, trash_dir)

```

### 6-3. `glob` (파일 목록 추출)

> **정의:** 윈도우 탐색기 검색창처럼 와일드카드(`*`, `?`)를 써서 파일 목록을 리스트로 가져옵니다.

```python
import glob
# 현재 폴더의 모든 txt 파일 찾기
print(glob.glob("*.txt"))

```

## 7. 데이터 포맷과 웹 (`json`, `urllib`)

### 7-1. `json` (매우 중요!)

> **정의:** 파이썬 데이터를 JSON 포맷(문자열/파일)으로 변환하거나, 그 반대로 변환합니다.

| 함수        | 방향     | 대상       | 설명                             |
| ----------- | -------- | ---------- | -------------------------------- |
| **`dumps`** | Py  JSON | **문자열** | 파이썬 객체를 JSON 문자열로 변환 |
| **`loads`** | JSON  Py | **문자열** | JSON 문자열을 파이썬 객체로 변환 |
| **`dump`**  | Py  JSON | **파일**   | 파일에 직접 저장                 |
| **`load`**  | JSON  Py | **파일**   | 파일에서 직접 읽기               |

```python
import json
data = {'name': '홍길동', 'age': 30}
# ensure_ascii=False: 한글 깨짐 방지
print(json.dumps(data, ensure_ascii=False, indent=4))

```

### 7-2. `urllib` & `webbrowser`

> **정의:** 웹 데이터를 다루거나 브라우저를 제어합니다.

* **`urllib`**: 코드로 웹 페이지 내용을 가져옵니다.
* **`webbrowser`**: 코드로 실제 웹 브라우저를 띄웁니다.

```python
import webbrowser
webbrowser.open("[http://google.com](http://google.com)")

```

## 8. 기타 필수 유틸리티

### 8-1. `pickle`

> **정의:** 파이썬의 객체(리스트, 클래스 등)를 **있는 그대로(바이너리)** 파일에 저장하고 복원합니다.

```python
import pickle
# 저장 (wb: 쓰기+바이너리)
pickle.dump([1, 2, 3], open("data.p", "wb"))
# 읽기 (rb: 읽기+바이너리)
data = pickle.load(open("data.p", "rb"))

```

### 8-2. `zipfile`

> **정의:** ZIP 파일을 생성하거나 압축을 해제합니다.

```python
import zipfile
# 파일 압축하기
with zipfile.ZipFile('archive.zip', 'w') as myzip:
    myzip.write('data.txt')

```

### 8-3. `threading`

> **정의:** 한 프로세스 안에서 여러 작업이 **동시에** 실행되게 합니다. (백그라운드 작업 등)

```python
import threading
def say(msg):
    while True: print(msg)

t = threading.Thread(target=say, args=("안녕",))
t.daemon = True # 메인 프로그램 종료 시 함께 종료
t.start()

```

### 8-4. `tempfile`

> **정의:** 임시로 파일을 만들어서 쓰고, 닫으면 자동으로 삭제되는 파일을 생성합니다. (보안/테스트용)

### 8-5. `traceback`

> **정의:** 프로그램이 에러로 죽지 않게 하면서, 에러의 상세 내용(스택 트레이스)을 확인하고 싶을 때 씁니다.

```python
import traceback
try:
    4 / 0
except:
    # 에러가 났지만 프로그램은 죽지 않고 로그만 남김
    print(traceback.format_exc()) 

```
