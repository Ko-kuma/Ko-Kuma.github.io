## 시작하기 전에..
사람은 누구에게나 '이름' = '홍길동', '생일' = '몇 월 며칠' 같은 대응 관계가 있습니다. 파이썬의 **딕셔너리(Dictionary)**는 이러한 **대응 관계**를 나타내는 자료형입니다.

* **리스트**: 순서가 있는 요소들의 모임 (인덱스 0, 1, 2... 로 접근)
* **딕셔너리**: **Key(키)**와 **Value(값)**의 쌍(Pair)으로 이루어짐 (순서가 없음)

## 1. 딕셔너리는 어떻게 만들까?

딕셔너리는 중괄호 `{}`로 감싸서 만들며, 각 요소는 `Key: Value` 형태를 띠고 쉼표(`,`)로 구분합니다.

```python
# 기본 구조: {Key1: Value1, Key2: Value2, ...}

# 1. 일반적인 딕셔너리
dic = {'name': 'pey', 'phone': '010-1234-5678', 'birth': '1118'}

# 2. Key에 정수, Value에 리스트도 가능
a = {1: 'hi', 'b': [1, 2, 3]}

# 3. 빈 딕셔너리
empty_dict = {}

```

---

## 2. 딕셔너리 쌍 추가하기 & 삭제하기

딕셔너리는 리스트처럼 값을 자유롭게 추가하거나 삭제할 수 있습니다. (**Mutable**)

### 1) 쌍 추가하기

`딕셔너리[Key] = Value` 형식으로 입력합니다.

```python
a = {1: 'a'}
a[2] = 'b'         # Key가 2, Value가 'b'인 쌍 추가
a['name'] = 'pey'  # Key가 'name', Value가 'pey'인 쌍 추가
a[3] = [1, 2, 3]   # Value에 리스트 추가

print(a)
# {1: 'a', 2: 'b', 'name': 'pey', 3: [1, 2, 3]}

```

### 2) 요소 삭제하기

`del 딕셔너리[Key]`를 사용하면 지정한 Key와 그에 해당하는 Value가 함께 삭제됩니다.

```python
del a[1]  # Key가 1인 요소 삭제
print(a)
# {2: 'b', 'name': 'pey', 3: [1, 2, 3]}

```

## 3. 딕셔너리 사용법 (Key를 사용해 Value 얻기)

리스트나 튜플은 요소값을 얻을 때 인덱싱(0, 1...)을 사용하지만, 딕셔너리는 **오직 Key를 통해서만 Value를 얻을 수 있습니다.**

```python
grade = {'pey': 10, 'julliet': 99}

print(grade['pey'])     # 10
print(grade['julliet']) # 99

```

## 4. 딕셔너리 만들 때 주의할 사항

딕셔너리를 사용할 때 다음 두 가지는 꼭 기억해야 합니다.

### 1) Key는 고유해야 한다 (중복 불가)

Key가 중복되면 **하나를 제외한 나머지 모든 Key:Value 쌍이 무시**됩니다. (어떤 것이 남을지는 알 수 없음)

```python
a = {1: 'a', 1: 'b'}
print(a) # {1: 'b'} -> 앞의 1:'a'가 무시됨

```

### 2) Key에는 '변하지 않는 값'만 올 수 있다

* **Key**: 불변(Immutable) 값인 문자열, 정수, 튜플 등은 가능하지만, **리스트(List)는 사용할 수 없습니다.** (리스트는 값이 변할 수 있기 때문)
* **Value**: 아무 값이나 상관없습니다. (리스트, 딕셔너리 등 모두 가능)

```python
# a = {[1, 2]: 'hi'} 
# -> 오류 발생: TypeError: unhashable type: 'list'

```

## 5. 딕셔너리 관련 함수 (Methods)

### 1) Key 리스트 만들기 (`keys`)

딕셔너리의 Key들만 모아서 반환합니다.

```python
a = {'name': 'pey', 'phone': '0119993323', 'birth': '1118'}
print(a.keys())
# dict_keys(['name', 'phone', 'birth'])

```

> **파이썬 3.0 이상에서의 변경점**
> * 파이썬 2.7까지는 `a.keys()`가 실제 리스트(`List`)를 반환했습니다.
> * **파이썬 3.0부터는** 메모리 낭비를 줄이기 위해 `dict_keys`라는 객체를 반환합니다.
> * 만약 리스트 고유의 기능(인덱싱 등)이 필요하다면 `list(a.keys())`로 변환해서 사용해야 합니다.
> 
> 

```python
# 리스트로 변환하기
key_list = list(a.keys())
print(key_list) # ['name', 'phone', 'birth']

```

### 2) Value 리스트 만들기 (`values`)

Key와 마찬가지로 Value만 모아서 `dict_values` 객체로 반환합니다.

```python
print(a.values())
# dict_values(['pey', '0119993323', '1118'])

```

### 3) Key, Value 쌍 얻기 (`items`)

Key와 Value를 튜플로 묶은 `dict_items` 객체를 반환합니다.

```python
print(a.items())
# dict_items([('name', 'pey'), ('phone', '0119993323'), ('birth', '1118')])

```

### 4) Key: Value 쌍 모두 지우기 (`clear`)

딕셔너리 안의 모든 요소를 삭제하여 빈 딕셔너리로 만듭니다.

```python
a.clear()
print(a) # {}

```

### 5) Key로 Value 얻기 (`get`)

`a['key']`와 동일한 기능을 하지만, **Key가 없을 때의 동작**이 다릅니다.

* `a['nokey']`: 오류(KeyError) 발생
* `a.get('nokey')`: **`None` 반환 (오류 발생 X)**
* `a.get('nokey', 'default')`: Key가 없을 때 미리 정해둔 **기본값('default')**을 반환

```python
a = {'name': 'pey', 'phone': '0119993323'}

print(a.get('name'))        # 'pey'
print(a.get('address'))     # None
print(a.get('address', 'Seoul')) # 'Seoul' (기본값 반환)

```

### 6) 해당 Key가 딕셔너리 안에 있는지 조사하기 (`in`)

특정 Key가 딕셔너리에 존재하면 `True`, 없으면 `False`를 반환합니다.

```python
a = {'name': 'pey', 'phone': '0119993323'}

print('name' in a)  # True
print('email' in a) # False

```
