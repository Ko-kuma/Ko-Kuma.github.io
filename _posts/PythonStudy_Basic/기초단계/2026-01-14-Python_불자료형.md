## 시작하기 전에..

**불(Bool)** 자료형은 오직 **참(True)**과 **거짓(False)** 두 가지 값만을 가질 수 있는 데이터 타입입니다. 주로 흐름 제어(if, while)나 논리 연산에 사용됩니다.

## 1. 불 자료형은 어떻게 사용할까?

파이썬에서 불 자료형을 사용할 때는 반드시 **첫 글자를 대문자**로 써야 합니다. (소문자 `true`는 변수명으로 인식하여 에러가 발생합니다.)

```python
# 1. 직접 대입
a = True
b = False

# 2. 조건문이나 비교 연산의 결과
print(1 == 1)  # True (같다)
print(2 > 1)   # True (크다)
print(2 < 1)   # False (작다)

# 자료형 확인
print(type(a)) # <class 'bool'>

```

---

## 2. 자료형의 참과 거짓 (Truthiness)

파이썬의 독특하고 강력한 특징 중 하나는 **숫자나 문자열, 리스트 등도 참과 거짓으로 판별**한다는 것입니다.

기본 원칙은 **"비어있으면 거짓(False), 값이 있으면 참(True)"**입니다.

### [자료형별 참/거짓 기준표]

| 자료형       | 참 (True)                    | 거짓 (False)       |
| ------------ | ---------------------------- | ------------------ |
| **문자열**   | `"python"` (문자가 있음)     | `""` (빈 문자열)   |
| **리스트**   | `[1, 2, 3]`                  | `[]` (빈 리스트)   |
| **튜플**     | `(1, 2)`                     | `()` (빈 튜플)     |
| **딕셔너리** | `{'a': 1}`                   | `{}` (빈 딕셔너리) |
| **숫자형**   | `1`, `-1` (0이 아닌 모든 수) | `0`                |
| **None**     | -                            | `None`             |

### 실제 활용 예제

이 특징을 이용하면 코드를 매우 간결하게 짤 수 있습니다.

```python
# 리스트가 비어있지 않으면 계속 꺼내기
a = [1, 2, 3, 4]
while a:            # a가 참인 동안 (즉, 요소가 남아있는 동안)
    print(a.pop())  # 요소를 하나씩 꺼냄

# 결과: 4, 3, 2, 1 순서로 출력되고, a가 빈 리스트([])가 되면 반복 종료

```

## 3. 불 연산 (논리 연산자)

참과 거짓을 가지고 연산을 수행하는 3가지 주요 연산자가 있습니다.

* **`not`**: 반대로 뒤집기 (True -> False, False -> True)
* **`and`**: 둘 다 참이어야 참 (교집합 개념)
* **`or`**: 둘 중 하나만 참이어도 참 (합집합 개념)

```python
x = True
y = False

# 1. not 연산
print(not x)      # False (True의 반대)
print(not y)      # True (False의 반대)

# 2. and 연산
print(x and y)    # False (하나가 거짓이므로)
print(x and True) # True (둘 다 참)

# 3. or 연산
print(x or y)     # True (x가 참이므로)
print(False or False) # False

```

### `bool()` 함수 사용하기

특정 데이터가 참인지 거짓인지 헷갈릴 때는 `bool()` 함수를 사용해 확인할 수 있습니다.

```python
print(bool('python')) # True
print(bool(''))       # False
print(bool(0))        # False
print(bool(3))        # True

```
