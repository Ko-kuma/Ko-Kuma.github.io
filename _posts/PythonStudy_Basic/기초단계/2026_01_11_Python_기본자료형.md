파이썬 프로그래밍의 기초인 **숫자형(Number)** 데이터 타입과 이를 다루는 다양한 **연산자**에 대해 정리합니다.

## 1. 숫자형 (Number) 종류와 표현 범위

파이썬에서 숫자를 표현하는 자료형은 크게 정수, 실수, 복소수로 나뉩니다.

### 1) 정수형 (Integer, `int`)
양의 정수, 음의 정수, 0을 포함합니다.
* **표현 범위**: **제한 없음** (메모리가 허용하는 한 무한대의 정수 표현 가능)
* **특징**: C언어나 Java의 `long` 타입(약 900경까지 표현)과 달리, 파이썬 3의 정수는 오버플로우(Overflow)가 발생하지 않아 천문학적인 숫자도 문제없이 계산할 수 있습니다.

### 2) 실수형 (Floating-point, `float`)
소수점이 포함된 숫자입니다.
* **표현 범위**: 약 $\pm 1.8 \times 10^{308}$ (매우 큰 범위)
* **정밀도**: IEEE 754 표준의 64비트 배정밀도(Double Precision)를 따르며, 소수점 아래 약 **15~17자리**까지 정확하게 표현합니다.
* **주의**: 컴퓨터의 실수 표현 방식상 미세한 오차가 발생할 수 있습니다. (예: `0.1 + 0.2` 가 `0.30000000000000004`가 되는 현상)

### 3) 복소수 (Complex, `complex`)
실수부와 허수부(`j`)로 이루어진 수입니다.

### 4) 숫자 가독성을 위한 팁 (`_`)
숫자의 단위가 커질 경우, 쉼표(`,`) 대신 언더스코어(`_`)를 사용하여 자릿수를 구분할 수 있습니다. 파이썬은 이를 무시하고 숫자로 인식합니다.

```python
# 숫자형 예제 및 범위 확인
import sys

# 정수: 제한 없이 큰 수 저장 가능
huge_int = 1234567890123456789012345678901234567890
print(huge_int)

# 실수: 최대값 확인
print(sys.float_info.max) # 약 1.79e+308

# 복소수
c = 1 + 2j

# 가독성을 위한 언더스코어
money = 10_000_000 # 10,000,000

```

## 2. 진수 표현 (8진수, 16진수, 2진수)

10진수 외의 숫자를 표현할 때는 접두어를 사용합니다.

* **2진수 (Binary)**: `0b` (숫자 0 + 알파벳 b)
* **8진수 (Octal)**: `0o` (숫자 0 + 알파벳 o)
* **16진수 (Hexadecimal)**: `0x` (숫자 0 + 알파벳 x)

```python
# 진수 표현 예제
bin_num = 0b101    # 2진수 (10진수 값: 5)
oct_num = 0o177    # 8진수 (10진수 값: 127)
hex_num = 0x8ff    # 16진수 (10진수 값: 2303)

print(bin_num, oct_num, hex_num)

```

## 3. 숫자형 연산자 (Operators)

파이썬에서 계산을 위해 사용하는 주요 연산자입니다.

### 1) 사칙연산 (+, -, *, /)

가장 기본이 되는 연산자입니다.

> **주의**: 파이썬 3부터는 나눗셈(`/`)의 결과가 나누어 떨어지더라도 항상 **실수형(float)**으로 반환됩니다. (예: `4 / 2`  `2.0`)

### 2) 제곱, 몫, 나머지 (**, //, %)

알고리즘 문제 풀이나 실무에서 매우 자주 사용되는 연산자들입니다.

* `**`: x의 y제곱을 리턴 ()
* `//`: 나눗셈 후 **몫(정수 부분)**만 리턴
* `%`: 나눗셈 후 **나머지**를 리턴

```python
x = 9
y = 2

# 사칙연산
print(x + y)  # 11
print(x / y)  # 4.5 (항상 float 반환)

# 특수 연산자
print(x ** y) # 81 (9의 2승)
print(x // y) # 4 (몫)
print(x % y)  # 1 (나머지)

```

## 4. [심화] 복합 대입 연산자 & 형 변환

### 1) 복합 대입 연산자 (Augmented Assignment)

변수의 값을 변경하고 다시 그 변수에 저장할 때 사용하는 단축 문법입니다.

| 연산자 | 설명        | 동일한 표현 |
| ------ | ----------- | ----------- |
| `+=`   | 더해서 대입 | `a = a + 1` |
| `-=`   | 빼서 대입   | `a = a - 1` |
| `*=`   | 곱해서 대입 | `a = a * 2` |
| `/=`   | 나눠서 대입 | `a = a / 2` |

```python
num = 10
num += 5   # num은 15가 됨
num *= 2   # num은 30이 됨 (15 * 2)
print(num)

```

### 2) 형 변환 (Type Casting)

데이터의 타입을 강제로 변경해야 할 때 사용합니다.

* `int()`: 정수로 변환 (실수를 넣으면 소수점 버림)
* `float()`: 실수로 변환
* `str()`: 문자열로 변환

```python
pi = 3.14159
str_num = "100"

print(int(pi))      # 3 (소수점 절삭)
print(float(5))     # 5.0
print(int(str_num)) # 100 (문자열 -> 정수)
```
