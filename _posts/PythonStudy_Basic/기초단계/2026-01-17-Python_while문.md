# 시작하기 전에..
조건문(`if`)이 상황에 따른 **판단**이라면, 반복문(`while`)은 **꾸준한 실행**입니다. 
특정 조건이 만족할 때까지 계속해서 문장을 수행하도록 만드는 `while`문에 대해 알아봅니다.

## 1. while문의 기본 구조

`while`문은 조건문이 **참(True)**인 동안, 아래에 속한 문장들을 계속해서 반복 수행합니다.

```python
while 조건문:
    수행할_문장1
    수행할_문장2
    수행할_문장3

```

* **동작 원리:** 조건문을 확인하고 참이면 수행할 문장을 실행합니다. 블록의 끝에 도달하면 다시 위로 올라가 조건문을 확인합니다. 조건이 거짓(False)이 되면 반복을 멈춥니다.

## 2. while문 만들기 (예제: 열 번 찍어 안 넘어가는 나무 없다)

가장 대표적인 예제로 `while`문의 흐름을 이해해 봅시다.

```python
treeHit = 0

while treeHit < 10:
    treeHit = treeHit + 1  # 찍은 횟수 1 증가
    print(f"나무를 {treeHit}번 찍었습니다.")
    
    if treeHit == 10:
        print("나무 넘어갑니다.")

```

**[실행 과정]**

1. `treeHit`이 0이므로 `treeHit < 10` 조건은 **참**입니다.
2. `treeHit`을 1 증가시키고 메시지를 출력합니다.
3. 다시 `while` 조건으로 돌아갑니다.
4. ... 반복 ...
5. `treeHit`이 10이 되면 메시지를 출력하고, 다시 조건으로 돌아갑니다.
6. `10 < 10`은 **거짓**이므로 `while`문을 빠져나옵니다.

## 3. while문 강제로 빠져나가기 (break문)

반복문이 돌고 있는 도중에, 조건과 상관없이 강제로 멈추고 싶을 때가 있습니다. 이때 사용하는 것이 `break`입니다.

### break문이란?

> "반복문을 깨뜨리고(break) 즉시 탈출하라"는 명령어입니다.

**예제: 커피 자판기**
커피가 다 떨어지면, 돈이 아무리 많아도 판매를 중단해야 합니다.

```python
coffee = 10
money = 300

while money:  # money가 0이 아니면 항상 True
    print("돈을 받았으니 커피를 줍니다.")
    coffee = coffee - 1
    print(f"남은 커피의 양은 {coffee}개입니다.")
    
    if coffee == 0:
        print("커피가 다 떨어졌습니다. 판매를 중지합니다.")
        break  # while문을 즉시 탈출!

```

## 4. while문 맨 처음으로 돌아가기 (continue문)

반복문을 빠져나가는 것이 아니라, **이번 차례만 건너뛰고** 다시 조건문 검사 부분으로 돌아가고 싶을 때는 `continue`를 사용합니다.

**예제: 1부터 10까지의 숫자 중 홀수만 출력하기**

```python
a = 0
while a < 10:
    a = a + 1
    if a % 2 == 0: 
        continue  # 짝수라면? 아래 print를 실행하지 않고 while 조건문으로 점프!
    print(a)

```

* `continue`를 만나면 그 아래에 있는 코드는 무시되고, 루프의 맨 위(조건 확인)로 돌아갑니다.

## 5. 무한 루프 (Infinite Loop)

무한 루프란 프로그램이 끝없이 계속 반복되는 상태를 말합니다. 서버나 대기 상태의 프로그램에서 자주 사용됩니다.

**기본 구조**
조건문 자리에 항상 참이 되는 값(`True`)을 넣습니다.

```python
while True:
    print("Ctrl+C를 눌러야 나갈 수 있습니다.")

```

> **주의:** IDLE이나 터미널에서 무한 루프를 실행했다면, **[Ctrl + C]** 키를 눌러서 강제로 종료해야 합니다. (KeyboardInterrupt 발생)
