# Python 제어문 문제풀이 (if, for, while) - 초급

초급자를 위한 if, for, while문 문제와 상세한 풀이입니다.

---

## 1️⃣ if문 문제

### 문제 1-1: 숫자 비교
**문제**: 두 숫자를 입력받아 더 큰 숫자를 출력하세요.

```python
a = 10
b = 20

if a > b:
    print(f"{a}가 더 큽니다.")
else:
    print(f"{b}가 더 큽니다.")

# 출력: 20가 더 큽니다.
```

**설명**:
- `if a > b:` - a가 b보다 크면 첫 번째 블록 실행
- `else:` - 조건이 거짓이면 두 번째 블록 실행

---

### 문제 1-2: 홀수/짝수 판별
**문제**: 입력받은 숫자가 홀수인지 짝수인지 판별하세요.

```python
num = 15

if num % 2 == 0:
    print(f"{num}은 짝수입니다.")
else:
    print(f"{num}은 홀수입니다.")

# 출력: 15은 홀수입니다.
```

**설명**:
- `num % 2 == 0` - 2로 나눈 나머지가 0이면 짝수
- `%` 연산자는 나머지를 구합니다

---

### 문제 1-3: 학점 계산
**문제**: 점수를 입력받아 학점을 출력하세요.
- 90점 이상: A
- 80점 이상: B
- 70점 이상: C
- 그 외: F

```python
score = 85

if score >= 90:
    print("학점: A")
elif score >= 80:
    print("학점: B")
elif score >= 70:
    print("학점: C")
else:
    print("학점: F")

# 출력: 학점: B
```

**설명**:
- `elif` - 여러 조건을 순차적으로 확인
- 맨 위에서부터 조건을 확인하여 참인 첫 번째 블록만 실행

---

### 문제 1-4: 나이 확인
**문제**: 나이를 입력받아 성인인지 미성년자인지 판별하세요. (18세 이상: 성인)

```python
age = 20

if age >= 18:
    print("성인입니다.")
else:
    print("미성년자입니다.")

# 출력: 성인입니다.
```

---

### 문제 1-5: 세 수 중 최댓값 찾기
**문제**: 세 개의 숫자 중 가장 큰 수를 찾으세요.

```python
a, b, c = 10, 25, 15

if a > b and a > c:
    max_num = a
elif b > a and b > c:
    max_num = b
else:
    max_num = c

print(f"최댓값: {max_num}")

# 출력: 최댓값: 25
```

**설명**:
- `and` - 모든 조건이 참이어야 함
- 논리 연산자를 사용하여 복합 조건 생성

## 2️⃣ for문 문제

### 문제 2-1: 1부터 5까지 출력
**문제**: for문을 사용하여 1부터 5까지 출력하세요.

```python
for i in range(1, 6):
    print(i)

# 출력:
# 1
# 2
# 3
# 4
# 5
```

**설명**:
- `range(1, 6)` - 1부터 5까지 (6은 포함 안 함)
- `range(시작, 끝)` - 시작부터 끝-1까지 생성

---

### 문제 2-2: 구구단 (2단)
**문제**: 2단 구구단을 출력하세요.

```python
for i in range(1, 10):
    print(f"2 × {i} = {2 * i}")

# 출력:
# 2 × 1 = 2
# 2 × 2 = 4
# 2 × 3 = 6
# ...
# 2 × 9 = 18
```

---

### 문제 2-3: 리스트 순회
**문제**: 주어진 리스트의 모든 요소를 출력하세요.

```python
fruits = ["사과", "바나나", "포도", "딸기"]

for fruit in fruits:
    print(fruit)

# 출력:
# 사과
# 바나나
# 포도
# 딸기
```

**설명**:
- `for 변수 in 리스트:` - 리스트의 각 요소를 순회
- 간단하고 직관적인 반복

---

### 문제 2-4: 합계 계산
**문제**: 1부터 10까지의 합계를 구하세요.

```python
total = 0

for i in range(1, 11):
    total += i

print(f"합계: {total}")

# 출력: 합계: 55
```

**설명**:
- `total += i` - total = total + i와 동일
- 누적 계산을 위해 초기값 설정 필수

---

### 문제 2-5: 구구단 전체 (이중 for문)
**문제**: 2단부터 9단까지의 구구단을 출력하세요.

```python
for dan in range(2, 10):
    for num in range(1, 10):
        print(f"{dan} × {num} = {dan * num}", end="  ")
    print()  # 줄바꿈

# 출력:
# 2 × 1 = 2  2 × 2 = 4  2 × 3 = 6  ...
# 3 × 1 = 3  3 × 2 = 6  3 × 3 = 9  ...
# ...
```

**설명**:
- 이중 for문을 사용하여 중첩된 반복
- `end="  "` - 줄바꿈 대신 공백 출력
- `print()` - 줄바꿈

### 문제 2-6: 개수 세기
**문제**: 리스트에서 3보다 큰 숫자의 개수를 세세요.

```python
numbers = [1, 2, 3, 4, 5, 6, 7]
count = 0

for num in numbers:
    if num > 3:
        count += 1

print(f"3보다 큰 숫자의 개수: {count}")

# 출력: 3보다 큰 숫자의 개수: 4
```

## 3️⃣ while문 문제

### 문제 3-1: 1부터 5까지 출력
**문제**: while문을 사용하여 1부터 5까지 출력하세요.

```python
i = 1

while i <= 5:
    print(i)
    i += 1

# 출력:
# 1
# 2
# 3
# 4
# 5
```

**설명**:
- `while 조건:` - 조건이 참인 동안 반복
- `i += 1` - 반드시 증가시키지 않으면 무한 루프

---

### 문제 3-2: 숫자 맞추기
**문제**: 정해진 숫자(예: 5)를 맞추는 게임. 입력받은 숫자가 정답과 같을 때까지 반복하세요.

```python
answer = 5
guess = 0

while guess != answer:
    guess = int(input("숫자를 맞춰보세요 (1-10): "))
    if guess < answer:
        print("더 큰 숫자입니다.")
    elif guess > answer:
        print("더 작은 숫자입니다.")
    else:
        print("정답입니다!")

# 실행 예:
# 숫자를 맞춰보세요 (1-10): 3
# 더 큰 숫자입니다.
# 숫자를 맞춰보세요 (1-10): 7
# 더 작은 숫자입니다.
# 숫자를 맞춰보세요 (1-10): 5
# 정답입니다!
```

---

### 문제 3-3: 합계 계산
**문제**: 입력받은 숫자들의 합계를 구하세요. (음수 입력 시 종료)

```python
total = 0

while True:
    num = int(input("숫자를 입력하세요 (음수 입력 시 종료): "))
    
    if num < 0:
        break
    
    total += num

print(f"합계: {total}")

# 실행 예:
# 숫자를 입력하세요 (음수 입력 시 종료): 10
# 숫자를 입력하세요 (음수 입력 시 종료): 20
# 숫자를 입력하세요 (음수 입력 시 종료): 30
# 숫자를 입력하세요 (음수 입력 시 종료): -1
# 합계: 60
```

**설명**:
- `break` - while 루프를 즉시 종료
- `while True:` - 무한 루프 (break로 빠져나옴)

---

### 문제 3-4: 팩토리얼 계산
**문제**: 입력받은 숫자의 팩토리얼을 계산하세요.

```python
n = 5
factorial = 1
i = 1

while i <= n:
    factorial *= i
    i += 1

print(f"{n}! = {factorial}")

# 출력: 5! = 120
```

**설명**:
- 팩토리얼(5!) = 5 × 4 × 3 × 2 × 1 = 120
- `factorial *= i` - factorial = factorial * i와 동일

## 4️⃣ 혼합 문제

```python
for i in range(1, 21):

```

---

### 문제 4-2: if와 while 함께 사용
**문제**: 1부터 100까지 더한 후, 합계가 500을 넘으면 종료하세요.

```python
total = 0
i = 1

while True:
    total += i
    
    if total > 500:
        print(f"{i}를 더했을 때 합계({total})가 500을 넘었습니다.")
        break
    
    i += 1

# 출력: 33를 더했을 때 합계(561)가 500을 넘었습니다.
```

---

### 문제 4-3: for, while, if 모두 사용
**문제**: 1부터 3단 구구단까지 출력하되, 결과가 10 이상인 것만 출력하세요.

```python
for dan in range(1, 4):
    num = 1
    while num <= 9:
        result = dan * num
        if result >= 10:
            print(f"{dan} × {num} = {result}")
        num += 1

# 출력:
# 2 × 5 = 10
# 2 × 6 = 12
# 2 × 7 = 14
# 2 × 8 = 16
# 2 × 9 = 18
# 3 × 4 = 12
# 3 × 5 = 15
# ...
```

## 📝 주요 정리

| 제어문 | 용도 | 예시 |
|--------|------|------|
| **if** | 조건 판별 | `if x > 10:` |
| **elif** | 여러 조건 | `elif x < 5:` |
| **else** | 나머지 경우 | `else:` |
| **for** | 반복 (횟수 정함) | `for i in range(10):` |
| **while** | 반복 (조건 기반) | `while x < 10:` |
| **break** | 루프 종료 | 루프 내부에서 사용 |
| **continue** | 다음 반복으로 | 현재 반복 건너뜀 |
