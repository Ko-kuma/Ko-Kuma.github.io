문자열이 '문자들의 모임'이었다면, **리스트(List)**는 **'어떤 데이터든 모아서 관리할 수 있는 그릇'**입니다. 숫자, 문자열, 심지어 리스트 안에 리스트를 담을 수도 있습니다.

## 1. 리스트 만들기

리스트는 대괄호(`[]`)로 감싸고, 쉼표(`,`)로 요소를 구분합니다.

```python
# 빈 리스트
a = []
b = list()

# 데이터가 있는 리스트
c = [1, 2, 3]
d = ["Life", "is", "too", "short"]
e = [1, 2, "Life", "is"] # 숫자와 문자를 섞어서 저장 가능
f = [1, 2, ["Life", "is"]] # 리스트 안에 리스트 포함 (중첩)

```

---

## 2. 리스트의 인덱싱과 슬라이싱

리스트도 문자열처럼 **순서(인덱스)**가 있습니다. (0부터 시작)

### 1) 기본 인덱싱

```python
a = [1, 2, 3]
print(a[0])    # 1
print(a[0] + a[2]) # 1 + 3 = 4
print(a[-1])   # 3 (마지막 요소)

```

### 2) 삼중 중첩 리스트 인덱싱 (Triple Nested List)

리스트 안에 리스트, 그 안에 또 리스트가 있는 경우입니다. `[]`를 순서대로 파고든다고 생각하면 쉽습니다.

```python
# 구조: [1, 2, ['a', 'b', ['Life', 'is']]]
a = [1, 2, ['a', 'b', ['Life', 'is']]]

# 단계별 접근
print(a[2])          # ['a', 'b', ['Life', 'is']] (큰 리스트)
print(a[2][2])       # ['Life', 'is'] (작은 리스트)
print(a[2][2][0])    # 'Life' (최종 문자열)

```

### 3) 리스트 슬라이싱

리스트를 자를 때 사용합니다.

```python
a = [1, 2, 3, 4, 5]
print(a[0:2])  # [1, 2] (끝번호 포함 X)

```

### 4) 중첩 리스트 슬라이싱

중첩된 리스트 내부를 슬라이싱할 수도 있습니다.

```python
a = [1, 2, 3, ['a', 'b', 'c'], 4, 5]

# a[3]은 ['a', 'b', 'c'] 입니다.
print(a[3][:2])  # ['a', 'b']

```

## 3. 리스트 연산하기

리스트끼리 더하거나 반복할 수 있습니다.

```python
a = [1, 2, 3]
b = [4, 5, 6]

# 1. 리스트 더하기 (+)
print(a + b)  # [1, 2, 3, 4, 5, 6] -> 두 리스트가 합쳐짐

# 2. 리스트 반복하기 (*)
print(a * 3)  # [1, 2, 3, 1, 2, 3, 1, 2, 3]

# 3. 리스트 길이 구하기 (len)
print(len(a)) # 3

```

## 4. 리스트의 수정과 삭제

문자열(String)과 달리 **리스트는 값을 변경하거나 삭제할 수 있습니다. (Mutable)**

### 1) 리스트 수정하기

인덱스로 접근해 값을 덮어씌웁니다.

```python
a = [1, 2, 3]
a[2] = 4
print(a)  # [1, 2, 4]

```

### 2) `del` 함수로 삭제하기

파이썬 내장 함수인 `del`을 사용하여 특정 인덱스의 값을 지웁니다.

```python
a = [1, 2, 3, 4, 5]
del a[1]      # 인덱스 1번(값 2) 삭제
print(a)      # [1, 3, 4, 5]

del a[2:]     # 슬라이싱 기법으로 여러 개 삭제
print(a)      # [1, 3]

```

## 5. 리스트 관련 함수 (Methods)

리스트 변수 뒤에 점(`.`)을 찍어 사용하는 유용한 함수들입니다.

### 1) 요소 추가 (`append`, `insert`, `extend`)

* `append(x)`: 리스트의 **맨 마지막**에 x를 추가
* `insert(idx, x)`: 특정 **인덱스(idx) 위치**에 x를 삽입
* `extend(List)`: 리스트 뒤에 **다른 리스트**를 붙여 확장

```python
a = [1, 2, 3]

a.append(4)     # [1, 2, 3, 4]
a.insert(0, 5)  # [5, 1, 2, 3, 4] (0번째 위치에 5 삽입)
a.extend([6, 7])# [5, 1, 2, 3, 4, 6, 7] (리스트 더하기와 비슷)

```

> **주의**: `append`에 리스트를 넣으면 리스트 자체가 하나의 요소로 들어갑니다. (`[1, 2, [3, 4]]`)

### 2) 정렬과 뒤집기 (`sort`, `reverse`)

* `sort()`: 리스트를 순서대로 정렬 (기본: 오름차순)
* `reverse()`: 리스트를 역순으로 뒤집음 (정렬 아님, 순서만 반대로)

```python
a = [1, 4, 3, 2]

a.sort()
print(a)    # [1, 2, 3, 4]

a.reverse()
print(a)    # [4, 3, 2, 1]

```

### 3) 위치 찾기 및 개수 세기 (`index`, `count`)

* `index(x)`: x 값이 있는 **위치(인덱스)** 반환 (없으면 에러)
* `count(x)`: 리스트 안에 x가 **몇 개** 있는지 반환

```python
a = [1, 2, 3, 1]

print(a.index(3))  # 2 (3은 2번 인덱스에 있음)
print(a.count(1))  # 2 (1은 리스트에 2개 있음)

```

### 4) 요소 끄집어내기 및 제거 (`pop`, `remove`)

* `pop()`: 리스트의 맨 마지막 요소를 **돌려주고 그 요소는 삭제**
* `remove(x)`: 리스트에서 **첫 번째로 나오는 x**를 삭제

```python
# pop 예제
a = [1, 2, 3]
print(a.pop())   # 3 리턴
print(a)         # [1, 2] (3이 사라짐)

# remove 예제
b = [1, 2, 3, 1, 2]
b.remove(3)      # 3을 삭제
print(b)         # [1, 2, 1, 2]
b.remove(1)      # 맨 앞의 1 삭제
print(b)         # [2, 1, 2]

```
