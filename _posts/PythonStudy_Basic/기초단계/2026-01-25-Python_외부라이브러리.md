파이썬 설치 시 기본으로 제공되는 '표준 라이브러리'만으로도 많은 것을 할 수 있지만, 파이썬의 진정한 강력함은 **외부 라이브러리**에서 나옵니다.
전 세계 개발자들이 만들어 놓은 방대한 기능을 내 프로젝트로 가져와 사용하는 방법을 알아봅니다.

---

## 1. 외부 라이브러리와 pip

### 1-1. 외부 라이브러리란?
파이썬 설치 시 기본으로 포함되지 않아 **별도로 설치해야만 사용할 수 있는 라이브러리**입니다. 데이터 분석(Pandas), 웹 개발(Django), 머신러닝(Scikit-learn) 등 다양한 분야의 패키지들이 존재합니다.

### 1-2. pip란? (Package Installer for Python)
파이썬의 외부 라이브러리(패키지)를 설치하고 관리해 주는 **패키지 관리자**입니다. 스마트폰의 '앱스토어'나 '플레이스토어' 같은 역할을 합니다.

### 1-3. pip 핵심 명령어 정리

터미널(명령 프롬프트)에서 아래 명령어들을 입력하여 사용합니다.

| 기능                     | 명령어                             | 예시                          |
| :----------------------- | :--------------------------------- | :---------------------------- |
| **패키지 설치**          | `pip install [패키지명]`           | `pip install Faker`           |
| **패키지 삭제**          | `pip uninstall [패키지명]`         | `pip uninstall Faker`         |
| **특정 버전 설치**       | `pip install [패키지명]==[버전]`   | `pip install Faker==3.0`      |
| **최신 버전 업그레이드** | `pip install --upgrade [패키지명]` | `pip install --upgrade Faker` |
| **설치된 패키지 확인**   | `pip list`                         | `pip list`                    |

## 2. 가짜 데이터 생성기: Faker

테스트를 하다 보면 "임의의 이름, 주소, 이메일이 100개쯤 필요한데..." 할 때가 있습니다. `Faker`는 이런 테스트용 가짜 데이터를 아주 쉽게 만들어줍니다.

### 2-1. Faker 설치 및 기본 사용

**설치:**
```bash
pip install Faker

```

**사용하기:**

```python
from faker import Faker

# 한국어('ko_KR') 설정을 가진 가짜 데이터 생성기 생성
fake = Faker('ko_KR')

print(fake.name())    # 가짜 이름 출력 (예: 김철수)
print(fake.address()) # 가짜 주소 출력

```

### 2-2. Faker 활용하기 (자주 쓰는 메서드)

| 분류       | 메서드           | 설명              | 예시 결과                   |
| ---------- | ---------------- | ----------------- | --------------------------- |
| **이름**   | `fake.name()`    | 전체 이름         | 이민수                      |
| **주소**   | `fake.address()` | 전체 주소         | 서울특별시 강남구 역삼로... |
| **텍스트** | `fake.text()`    | 임의의 문장       | 가나다라 마바사...          |
| **인터넷** | `fake.email()`   | 이메일 주소       | example@email.com           |
| **회사**   | `fake.company()` | 회사명            | (주) 다이내믹코리아         |
| **날짜**   | `fake.date()`    | 날짜              | 2023-10-25                  |
| **프로필** | `fake.profile()` | 개인정보 딕셔너리 | `{'name': '...', ...}`      |

## 3. 방정식 기호 연산: SymPy

파이썬으로 수학 숙제를 할 수 있을까요?
`SymPy`는 숫자를 근사치로 계산하는 것이 아니라, ,  같은 **기호(Symbol)**를 사용하여 방정식을 풀거나 미분, 적분을 할 수 있게 해주는 라이브러리입니다.

### 3-1. SymPy 설치 및 사용해보기

**설치:**

```bash
pip install sympy

```

**기본 사용 (방정식 풀기):**
 의 해를 구해봅시다.

```python
from sympy import symbols, solve

x = symbols('x')   # x를 기호로 정의
equation = 2*x - 6 # 방정식 정의 (결과가 0이 되는 식)

result = solve(equation) 
print(result) 
# 결과: [3]

```

### 3-2. 여러 개의 기호 사용하기

 뿐만 아니라  등 여러 기호가 필요할 때가 있습니다.

```python
from sympy import symbols

x, y = symbols('x y') # 띄어쓰기로 구분하여 정의
expr = x + 2*y
print(expr + x) # 2*x + 2*y (동류항끼리 계산됨)

```

### 3-3. 유리수 연산하기 (`fractions.Fraction`)

컴퓨터는 소수점 계산 시 미세한 오차가 발생할 수 있습니다. 정확한 분수 연산을 위해 파이썬 표준 라이브러리인 `fractions`를 SymPy와 함께 자주 사용합니다.

```python
from fractions import Fraction
import sympy

# 1/5 + 1/5 = 2/5 (0.4)
result = Fraction(1, 5) + Fraction(2, 5)

print(result) # 3/5 (정확히 분수 형태로 출력됨)
print(float(result)) # 0.6

```

*(참고: SymPy 내부에도 `Rational`이라는 분수 자료형이 있지만, 파이썬 표준인 `Fraction`도 많이 쓰입니다.)*

### 3-4. SymPy 활용하기 (방정식과 미분/적분)

| 기능            | 함수        | 예제 코드            | 수학적 의미   |
| --------------- | ----------- | -------------------- | ------------- |
| **방정식 풀기** | `solve`     | `solve(x**2 - 4)`    | 의 해 ()      |
| **미분**        | `diff`      | `diff(x**3, x)`      | 을 로 미분 () |
| **적분**        | `integrate` | `integrate(x**2, x)` | 을 로 적분 () |
| **전개**        | `expand`    | `expand((x+1)**2)`   | 전개 ()       |
| **인수분해**    | `factor`    | `factor(x**2 - 1)`   | 인수분해 ()   |
