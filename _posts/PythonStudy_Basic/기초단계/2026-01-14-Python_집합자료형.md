## 시작하기 전에..
파이썬의 **집합(Set)** 자료형은 수학에서 배운 집합의 개념을 프로그래밍적으로 구현한 것입니다. 리스트나 딕셔너리와는 다른 독특한 특징 덕분에 특정 상황(중복 제거, 교집합 구하기 등)에서 매우 유용하게 쓰입니다.

## 1. 집합 자료형은 어떻게 만들까?

집합은 `set()` 키워드를 사용하거나, 중괄호 `{}` 안에 값을 쉼표로 구분하여 만듭니다.

```python
# 1. set() 키워드 사용
s1 = set([1, 2, 3])
print(s1)  # {1, 2, 3}

# 2. 문자열을 입력하면?
s2 = set("Hello")
print(s2)  # {'e', 'H', 'l', 'o'} (순서가 바뀌고 'l'이 하나 줄어듦)

# 3. 중괄호 {} 사용 (딕셔너리와 비슷하지만 Key만 있는 형태)
s3 = {1, 2, 3}

```

> **주의할 점: 빈 집합 만들기**
> * `{}`만 사용하면 빈 집합이 아니라 **빈 딕셔너리(Dictionary)**가 생성됩니다.
> * 빈 집합을 만들 때는 반드시 **`s = set()`**을 사용해야 합니다.

## 2. 집합 자료형의 특징

집합 자료형에는 다른 자료형과 구별되는 아주 중요한 두 가지 특징이 있습니다.

### 1) 중복을 허용하지 않는다 (Unique)

집합에 같은 값을 여러 번 넣어도 자동으로 중복이 제거되어 하나만 남습니다. 이 특징을 이용해 **리스트의 중복을 제거**할 때 자주 사용합니다.

```python
l = [1, 1, 2, 2, 3, 4, 4, 5]
new_l = list(set(l))  # 리스트 -> 집합(중복제거) -> 다시 리스트
print(new_l) # [1, 2, 3, 4, 5]

```

### 2) 순서가 없다 (Unordered)

리스트나 튜플과 달리 **순서(인덱스)가 없습니다.** 따라서 `s1[0]`처럼 인덱싱으로 값을 얻을 수 없습니다.

* 만약 인덱싱이 필요하다면, `list()`나 `tuple()` 함수를 써서 변환한 후에 사용해야 합니다.

## 3. 교집합, 합집합, 차집합 구하기

집합 자료형을 사용하는 가장 큰 이유입니다. 두 집합 간의 관계를 아주 쉽게 구할 수 있습니다.

```python
s1 = set([1, 2, 3, 4, 5, 6])
s2 = set([4, 5, 6, 7, 8, 9])

```

### 1) 교집합 (Intersection)

두 집합에 공통으로 포함된 요소를 구합니다.

* 기호: `&`
* 함수: `.intersection()`

```python
print(s1 & s2)            # {4, 5, 6}
print(s1.intersection(s2))# {4, 5, 6}

```

### 2) 합집합 (Union)

두 집합의 모든 요소를 합칩니다. (중복은 자동 제거)

* 기호: `|` (Shift + 백슬래시)
* 함수: `.union()`

```python
print(s1 | s2)            # {1, 2, 3, 4, 5, 6, 7, 8, 9}
print(s1.union(s2))       # {1, 2, 3, 4, 5, 6, 7, 8, 9}

```

### 3) 차집합 (Difference)

한 집합에서 다른 집합의 요소를 뺍니다.

* 기호: `-`
* 함수: `.difference()`

```python
# s1에서 s2를 빼기 (1, 2, 3 남음)
print(s1 - s2)            # {1, 2, 3}
print(s1.difference(s2))  # {1, 2, 3}

# s2에서 s1을 빼기 (7, 8, 9 남음)
print(s2 - s1)            # {8, 9, 7}

```

## 4. 집합 자료형 관련 함수 (값 추가 및 삭제)

집합은 값을 변경할 수 있는(Mutable) 자료형입니다.

### 1) 값 1개 추가하기 (`add`)

이미 있는 값을 추가하면 변화가 없습니다.

```python
s1 = {1, 2, 3}
s1.add(4)
print(s1)  # {1, 2, 3, 4}

```

### 2) 값 여러 개 추가하기 (`update`)

리스트 등을 사용하여 여러 값을 한꺼번에 추가합니다.

```python
s1 = {1, 2, 3}
s1.update([4, 5, 6])
print(s1)  # {1, 2, 3, 4, 5, 6}

```

### 3) 특정 값 제거하기 (`remove`)

특정 값을 찾아서 제거합니다.

```python
s1 = {1, 2, 3}
s1.remove(2)
print(s1)  # {1, 3}

```

> **참고**: `remove`는 지우려는 값이 집합에 없으면 오류(KeyError)가 발생합니다.
