## 1. 평균 점수 구하기

### 문제 1
국어 80점, 영어 75점, 수학 55점일 때, 변수를 사용하여 이 세 과목의 평균 점수를 구하는 코드를 작성하세요.

### 문제 2
리스트에 저장된 점수들의 평균을 구하세요.
`scores = [70, 60, 55, 75, 95, 90, 80, 80, 85, 100]`

<br>

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이 1**
```python
kor = 80
eng = 75
math = 55

average = (kor + eng + math) / 3
print(average) 
# 결과: 70.0

```

**풀이 2**
파이썬 내장 함수 `sum()`(합계)과 `len()`(길이/개수)을 활용하면 편리합니다.

```python
scores = [70, 60, 55, 75, 95, 90, 80, 80, 85, 100]
average = sum(scores) / len(scores)
print(average)
# 결과: 79.0

```

</div>
</details>

## 3. 주민등록번호 나누기

### 문제 5

`pin = "881120-1068234"` 문자열을 하이픈(`-`)을 기준으로 나누어 앞자리와 뒷자리를 담은 리스트를 만드세요.

### 문제 6

위와 같은 `pin` 문자열에서 슬라이싱을 사용하여 생년월일 부분('881120')만 추출하세요.

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이 5**
`split` 함수를 사용합니다.

```python
pin = "881120-1068234"
result = pin.split("-")
print(result)
# 결과: ['881120', '1068234']

```

**풀이 6**
인덱스 0부터 6전까지 슬라이싱합니다.

```python
pin = "881120-1068234"
print(pin[:6])
# 결과: '881120'

```

## 4. 주민등록번호 인덱싱

`pin = "881120-1068234"`에서 성별을 나타내는 숫자(하이픈 바로 뒤 숫자)를 인덱싱으로 출력하세요.

### 문제 8

위 `pin` 변수에서 맨 마지막 숫자를 **음수 인덱싱**을 사용하여 출력하세요.

<details>

## 5. 문자열 바꾸기

### 문제 9

문자열 `a = "Hi Java"`에서 "Java"를 "Python"으로 바꾸어 출력하세요.

### 문제 10

`phone = "010-1234-5678"`에서 모든 하이픈(`-`)을 공백(`     `)으로 바꾸세요.

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이 9**
`replace(찾을값, 바꿀값)` 메서드를 사용합니다. (문자열은 불변 객체이므로 바뀐 새로운 문자열을 반환합니다.)

```python
a = "Hi Java"

# 결과: Hi Python

```

**풀이 10**

```python
phone = "010-1234-5678"
print(phone.replace("-", " "))
```

## 6. 리스트 역순 정렬하기

### 문제 11

리스트 `a = [1, 3, 5, 4, 2]`를 내림차순(큰 수부터 작은 수)으로 정렬하여 저장하세요.

### 문제 12

리스트 `['a', 'c', 'b']`를 정렬하지 않고, 현재 순서의 **정반대**로 뒤집으세요.

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이 11**
`sort` 메서드에 `reverse=True` 옵션을 줍니다.

```python
a = [1, 3, 5, 4, 2]
a.sort(reverse=True)
print(a)
# 결과: [5, 4, 3, 2, 1]

```

**풀이 12**
순서만 뒤집을 때는 `reverse()` 메서드를 사용합니다.

```python
a = ['a', 'c', 'b']
a.reverse()
print(a)
# 결과: ['b', 'c', 'a']

```

## 7. 리스트를 문자열로 만들기

### 문제 13

리스트 `['Life', 'is', 'too', 'short']`를 공백을 사이에 넣어 하나의 문자열 "Life is too short"로 만드세요.

### 문제 14

리스트 `['010', '1234', '5678']`을 하이픈(`-`)으로 연결하여 하나의 전화번호 문자열로 만드세요.

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이 13**
`join` 메서드를 사용합니다. 사용법은 `"구분자".join(리스트)` 입니다.

```python
words = ['Life', 'is', 'too', 'short']
result = " ".join(words)
print(result)

```

**풀이 14**

```python
nums = ['010', '1234', '5678']
result = "-".join(nums)
print(result)
# 결과: 010-1234-5678

```

## 8. 튜플 더하기

### 문제 15

튜플 `(1, 2, 3)`에 `(4, )`를 더하여 `(1, 2, 3, 4)`를 만드세요.

### 문제 16

튜플 `t1 = (1, 2, 3)`에 정수 `4`를 바로 더하려고 할 때(`t1 + 4`) 오류가 납니다. 이유와 해결 방법은 무엇인가요?

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이 15**
튜플끼리의 덧셈은 리스트처럼 연결(concatenation)을 의미합니다.

```python
t1 = (1, 2, 3)
t2 = (4, )
print(t1 + t2)

```

**풀이 16**
**이유:** 튜플은 튜플끼리만 더할 수 있는데, `4`는 정수(int)이기 때문입니다.
**해결:** 4를 튜플인 `(4, )` 형태로 만들어 더해야 합니다. (요소가 하나일 땐 콤마 필수)

```python
# (1, 2, 3) + (4, )

```

</details>

## 9. 딕셔너리 Key 문제

### 문제 17

`a = {'name': 'pey', 'phone': '010', 'birth': '1118'}`의 Key들만 모아서 `dict_keys` 객체 혹은 리스트로 출력하세요.

### 문제 18

위 딕셔너리 `a`에 'email'이라는 Key가 있는지 조사하는 코드를 작성하세요. (True/False 출력)

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이 17**
`keys()` 메서드를 사용합니다. 리스트로 보려면 `list()`로 감싸줍니다.

```python
a = {'name': 'pey', 'phone': '010', 'birth': '1118'}
print(a.keys()) 
# 또는 print(list(a.keys()))

```

**풀이 18**
`in` 연산자를 사용합니다.

```python
print('email' in a)
```

## 11. 리스트에서 중복 제거하기

### 문제 21

리스트 `a = [1, 1, 1, 2, 2, 3, 3, 3, 4, 4, 5]`에서 중복된 숫자를 제거하기 위해 어떤 자료형으로 변환하면 가장 쉬울까요?

### 문제 22

위 리스트 `a`의 중복을 제거하고 다시 리스트 형태로 만드세요.

<details>
<summary><strong>정답 및 풀이 확인하기</strong></summary>
<div markdown="1">

**풀이 21**
**집합(Set)** 자료형입니다. 집합은 중복을 허용하지 않는 특징이 있습니다.

**풀이 22**

```python
a = [1, 1, 1, 2, 2, 3, 3, 3, 4, 4, 5]
result = list(set(a))
print(result)
# 결과: [1, 2, 3, 4, 5] (순서는 보장되지 않을 수 있음)

```

### 문제 24

두 변수 `a = 3`, `b = 5`의 값을 서로 맞바꾸는(swap) 가장 파이썬다운 코드를 작성하세요.

**풀이 24**
