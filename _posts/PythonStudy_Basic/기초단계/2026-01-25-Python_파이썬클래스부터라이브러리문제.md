## 1. 클래스 (Class)

### 문제 1: 클래스 상속받고 메서드 추가하기 (1)
기존 `Car` 클래스를 상속받는 `FlyingCar` 클래스를 만드세요.
* `Car` 클래스는 `drive()` 메서드를 가지고 있습니다. ("운전 중" 출력)
* `FlyingCar` 클래스에 `fly()` 메서드를 추가하여 "하늘을 나는 중"을 출력하세요.

### 문제 2: 클래스 상속받고 메서드 추가하기 (2)
계산기 클래스 `Calculator`를 상속받는 `MaxLimitCalculator`를 만드세요.
* 기존 `add` 메서드는 그대로 사용합니다.
* 숫자 값이 100이 넘어가면 무조건 100을 반환하도록 `add` 메서드를 **오버라이딩(재정의)** 하세요.
* (힌트: 부모의 `add`를 호출한 뒤 값을 검사하거나, 그냥 새로 짜도 됩니다.)

### 1. 클래스 상속

```python
# 문제 1
class Car:
    def drive(self):
        print("운전 중")

class FlyingCar(Car):
    def fly(self):
        print("하늘을 나는 중")

# 문제 2
class Calculator:
    def add(self, val):
        self.value += val

class MaxLimitCalculator(Calculator):
    def add(self, val):
        self.value += val
        if self.value > 100:
            self.value = 100

```
