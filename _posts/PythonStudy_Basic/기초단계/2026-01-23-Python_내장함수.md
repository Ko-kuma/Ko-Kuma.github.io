# 시작하기 전에..
파이썬이 강력한 이유는 '이미 만들어진 도구'가 매우 많기 때문이며,
`import` 없이 언제 어디서나 바로 꺼내 쓸 수 있는 **내장 함수**들을 정리해 봅니다.



## 1. 내장 함수란? (Built-in Function)

파이썬 설치 시 기본적으로 포함된 함수들을 말합니다.
우리가 흔히 썼던 `print()`, `len()`, `type()` 등이 모두 여기에 속합니다. 별도의 라이브러리를 불러올 필요 없이 바로 사용할 수 있어 매우 편리합니다.

## 2. 주요 내장 함수 상세 정리

### 수치 연산 관련
* **`abs(x)`**: 절댓값을 반환합니다. (`abs(-3)` $\rightarrow$ `3`)
* **`divmod(a, b)`**: 몫과 나머지를 튜플로 반환합니다. (`divmod(7, 3)` $\rightarrow$ `(2, 1)`)
* **`max(iterable)` / `min(iterable)`**: 최댓값과 최솟값을 찾습니다.
* **`pow(x, y)`**: x의 y제곱을 구합니다. (`pow(2, 3)` $\rightarrow$ `8`)
* **`round(number, ndigits)`**: 숫자를 반올림합니다.
* **`sum(iterable)`**: 리스트나 튜플의 모든 요소의 합을 구합니다.

### 자료형 변환 및 확인
* **`int(x)`**: 정수로 변환합니다.
* **`str(x)`**: 문자열로 변환합니다.
* **`list(x)`**: 리스트로 변환합니다.
* **`tuple(x)`**: 튜플로 변환합니다.
* **`type(object)`**: 객체의 자료형(타입)을 알려줍니다.
* **`isinstance(object, class)`**: 객체가 해당 클래스의 인스턴스인지 확인합니다. (`True`/`False`)

### 문자열 및 코드 관련
* **`chr(i)`**: 아스키(ASCII) 코드값을 입력받아 해당 문자를 반환합니다. (`chr(97)` $\rightarrow$ `'a'`)
* **`ord(c)`**: 문자를 입력받아 아스키 코드값을 반환합니다. (`ord('a')` $\rightarrow$ `97`)
* **`eval(expression)`**: 문자열로 된 파이썬 식을 실행합니다. (`eval('1+2')` $\rightarrow$ `3`)
* **`oct(x)`**: 정수를 8진수 문자열로 바꿉니다.

### 시퀀스(리스트 등) 처리 관련
* **`all(iterable)`**: 모두 참(True)이면 True 반환 (하나라도 거짓이면 False).
* **`any(iterable)`**: 하나라도 참(True)이면 True 반환 (모두 거짓이면 False).
* **`dir(object)`**: 객체가 가지고 있는 변수나 함수 목록을 보여줍니다.
* **`enumerate(iterable)`**: (인덱스, 값) 형태의 객체를 반환합니다. `for`문과 자주 쓰입니다.
* **`filter(func, iterable)`**: 함수 조건에 맞는 요소만 걸러냅니다.
* **`len(s)`**: 길이(개수)를 반환합니다.
* **`map(func, iterable)`**: 모든 요소에 함수를 적용합니다.
* **`range()`**: 숫자 범위를 생성합니다.
    * `range(10)`: 0부터 9까지
    * `range(1, 11)`: 1부터 10까지
    * `range(1, 10, 2)`: 1부터 9까지 2씩 증가
* **`sorted(iterable)`**: 입력 데이터를 정렬한 **새로운 리스트**를 반환합니다. (원본 유지)
* **`zip(iterable*)`**: 동일한 개수의 자료형을 묶어줍니다.

### 입출력 관련
* **`input()`**: 사용자 입력을 받습니다.
* **`open(filename, mode)`**: 파일을 엽니다.
    * `w`: 쓰기 (덮어쓰기)
    * `r`: 읽기
    * `a`: 추가 (이어쓰기)
    * `b`: 바이너리 모드

## 3. 내장 함수 한눈에 보기 (요약 표)

| 함수명           | 설명                          | 예제 코드                        | 결과                 |
| :--------------- | :---------------------------- | :------------------------------- | :------------------- |
| **`abs`**        | 절댓값 반환                   | `abs(-3)`                        | `3`                  |
| **`all`**        | 모두 참이면 True              | `all([1, 2, 3])`                 | `True`               |
| **`any`**        | 하나라도 참이면 True          | `any([0, ""])`                   | `False`              |
| **`chr`**        | 아스키코드 $\rightarrow$ 문자 | `chr(97)`                        | `'a'`                |
| **`dir`**        | 객체 속성/메서드 목록         | `dir([])`                        | `['append', ...]`    |
| **`divmod`**     | 몫과 나머지                   | `divmod(7, 3)`                   | `(2, 1)`             |
| **`enumerate`**  | 인덱스와 값 반환              | `list(enumerate(['a','b']))`     | `[(0,'a'), (1,'b')]` |
| **`eval`**       | 문자열 식 실행                | `eval('1+2')`                    | `3`                  |
| **`filter`**     | 조건에 맞는 값 추출           | `filter(lambda x: x>0, [1, -1])` | `[1]`                |
| **`input`**      | 사용자 입력 받기              | `a = input()`                    | (입력값)             |
| **`int`**        | 정수로 변환                   | `int('3')`                       | `3`                  |
| **`isinstance`** | 자료형 일치 확인              | `isinstance(3, int)`             | `True`               |
| **`len`**        | 길이 구하기                   | `len("python")`                  | `6`                  |
| **`list`**       | 리스트 생성                   | `list("ab")`                     | `['a', 'b']`         |
| **`map`**        | 함수 일괄 적용                | `list(map(int, ['1', '2']))`     | `[1, 2]`             |
| **`max`**        | 최댓값                        | `max([1, 2, 3])`                 | `3`                  |
| **`min`**        | 최솟값                        | `min([1, 2, 3])`                 | `1`                  |
| **`oct`**        | 8진수 변환                    | `oct(8)`                         | `'0o10'`             |
| **`open`**       | 파일 열기                     | `open("a.txt", "w")`             | (파일객체)           |
| **`ord`**        | 문자 $\rightarrow$ 아스키코드 | `ord('a')`                       | `97`                 |
| **`pow`**        | 제곱 ($x^y$)                  | `pow(2, 3)`                      | `8`                  |
| **`range`**      | 숫자 범위 생성                | `list(range(0, 5))`              | `[0, 1, 2, 3, 4]`    |
| **`round`**      | 반올림                        | `round(4.6)`                     | `5`                  |
| **`sorted`**     | 정렬된 리스트 반환            | `sorted([3, 1, 2])`              | `[1, 2, 3]`          |
| **`str`**        | 문자열 변환                   | `str(3)`                         | `'3'`                |
| **`sum`**        | 합계 구하기                   | `sum([1, 2, 3])`                 | `6`                  |
| **`tuple`**      | 튜플 생성                     | `tuple("ab")`                    | `('a', 'b')`         |
| **`type`**       | 자료형 확인                   | `type([])`                       | `<class 'list'>`     |
| **`zip`**        | 자료 묶기                     | `list(zip([1,2], ['a','b']))`    | `[(1,'a'), (2,'b')]` |
