## 문제 설명
Hello World!를 출력하는 아주 기초적인 문제입니다. 파이썬의 출력 방식을 익히기에 좋습니다.

## 문제 풀이 
```python
print("Hello World!")

```

## 코드 설명

* `print()`: 파이썬에서 가장 기본이 되는 출력 함수입니다. 
* 큰따옴표(`" "`) 혹은 작은따옴표(`' '`)로 감싼 문자열을 콘솔에 출력합니다.
