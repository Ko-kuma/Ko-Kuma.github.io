## 문제 설명
(세 자리 수) × (세 자리 수)는 다음과 같은 과정을 통하여 이루어진다.
```bash
   472......(1)
X  385......(2)
----------------
   2360......(3)
  3776......(4)
 1416......(5)
-----------------
181720......(6)  
```

(1)과 (2)위치에 들어갈 세 자리 자연수가 주어질 때 (3), (4), (5), (6)위치에 들어갈 값을 구하는 프로그램을 작성하시오.
## 문제 풀이
[int형변환],[//몫연산자],[%나머지연산자]를 활용해서 풀어내면 되겠습니다.
```python
A = int(input())
B = int(input())
one = B % 10
ten = (B // 10) % 10
hundred = B // 100
print(A * one)
print(A * ten)
print(A * hundred)
print(A * B)
```
해당 방식 처럼 %10으로 1의자리, B//10몫을 %10으로 나눠서 십의자리, //100을 마지막 100의 자리 숫자를 구해 total값을 내시면 되겠습니다.

아래방식으로 풀이 하는 법도 있습니다. 아래 방식은 B의 값을 문자열로 유지시키고 각 자리수를 인덱싱하여 문제를 푸시면 되겠습니다. 
```python
A = int(input())
B = input()
print(A * int(B[2]))
print(A * int(B[1]))
print(A * int(B[0]))
print(A * int(B))
```
각 자릿수를 분리해내는 공식을 찾아내는데 시간이 걸렸습니다. 고민하다가 구글링을 통해 나머지와 연산자를 조합하던지, 리스트를 활용하라는 걸 참고하여 문제를 해결했습니다. 결국 문제를 모르면 모르는 걸 인정하고 검증된 공식이나 다른 사람이 어떤 식으로 접근했고 해결했는지를 찾아내 배워가는 것도 필요하다고 배울 수 있던거 같습니다.(하지만, 어느정도 스스로 고민은 하고 찾아봐야하긴 합니다.)

## 코드 설명
* 수학적 접근 (%, // 연산자 활용)
    * 일의 자리: B % 10 (10으로 나눈 나머지)
    * 십의 자리: (B // 10) % 10 (10으로 나눈 몫에서 다시 10으로 나눈나머지)
    * 백의 자리: B // 100 (100으로 나눈 몫)

* 문자열 인덱싱 접근 
    * B를 문자열 상태로 두면 B[0](백의 자리), B[1](십의 자리), B[2](일의 자리)로 바로 접근할 수 있습니다. 단, 계산 시에는 다시 int()로 형변환이 필요합니다.
