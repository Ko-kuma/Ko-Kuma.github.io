## 문제 설명
두 정수 A와B를 입력 받은 다음, A+B를 출력하는 프로그램을 작성하세요

## 문제 풀이
헷갈리지 말고 문제 설명을 잘 봐야합니다.
여기서 [두 정수 A와 B를 입력 받아]라는 설명이 있습니다. 입력이라는 말이 들어 갔으므로 일단 해당 문제에는 **input**함수가 사용된다는 풀이가 나옵니다.

기본적으로 생각하면
```python
a = int(input("값1"))
b = int(input("값2"))
print(a + b)
```
위의 예제처럼 문제를 풀면되지 않을까?
라는 생각을 할 수 있습니다.

여기서, 가독성을 조금 더 좋게 하기 위해 [map]아리는 내장함수와 [split]라는 문자열 나누기 함수를  사용해서 풀이 해보겠습니다
```python
a, b = map(int, input().split()) 
print(a+b)
```
* a, b 라는 변수 생성 
* a, b 변수에 [map]이라는 내장함수를 사용 [map(함수, 반복가능한것)]
* int는 함수, input은 반복가능한 객체 input을 활용해 문자열 1과 2를 반환 받고 1과 2는[.split()]로 인해 리스트(반복가능)로 반환
* print(a+b)를 이용해 a, b를 정수로 반환  
## 코드 설명
* `input()`: 사용자로부터 데이터를 입력받지만, 무조건 **문자열(String)**로 가져옵니다.
* `.split()`: 공백을 기준으로 문자열을 쪼개 리스트 형태로 만듭니다. (예: `"1 2"` → `["1", "2"]`)

* 여기서 `int`는 매우 중요한 역할을 합니다. 단순히 숫자를 뜻하는 키워드가 아니라, **데이터의 타입을 바꿔주는 '형변환(Type Casting)'** 도구입니다.

* **본질**은 파이썬에서 정수 객체를 생성하는 **클래스**입니다.
* **기능**은 `input()`으로 들어온 `"1"`은 계산을 할 수 없는 **문자**입니다. 이 문자열을 산술 연산이 가능한 **정수(Integer)**로 변환하는 과정을 거쳐야만 `a + b`가 가능해집니다.
* `map(함수, 반복가능한객체)` 형태를 가집니다.
* 리스트 안에 들어있는 각각의 문자열들을 하나씩 꺼내 `int` 클래스에 집어넣어 한꺼번에 정수로 형변환을 시켜줍니다.
