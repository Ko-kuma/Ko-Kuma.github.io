## 문제 설명
두 정수 A와B를 입력 받은 다음, A/B를 출력하는 프로그램을 작성하세요

## 문제 풀이
이번 문제 풀이는 1000번 A+B 문제를 응용한 문제 라고 생각하면 됩니다.
1000번 문제는 +연산자를 사용하였으면 이번문제는 /연산자를 사용하며 정수를 나타내는 int가 아닌 소수점을 나타내야 하기 때문에 실수 float를 사용하시면 되겠습니다.

기본적으로 생각하면
```python
a = int(input("값1"))
b = int(input("값2"))
print(a / b)
```

가독성 높이기 
```python
a, b = map(float, input().split()) 
print(a/b)
```

## 코드 설명
* `input()`: 사용자로부터 데이터를 입력받지만, 무조건 **문자열(String)**로 가져옵니다.
* `.split()`: 공백을 기준으로 문자열을 쪼개 리스트 형태로 만듭니다. (예: `"1 2"` → `["1", "2"]`)

* 여기서 `float`는 매우 중요한 역할을 합니다. 단순히 숫자를 뜻하는 키워드가 아니라, **데이터의 타입을 바꿔주는 '형변환(Type Casting)'** 도구입니다.

* **본질**은 파이썬에서 실수 객체를 생성하는 **클래스**입니다.
* **기능**은 `input()`으로 들어온 `"1"`은 계산을 할 수 없는 **문자**입니다. 이 문자열을 산술 연산이 가능한 **실수(float)**로 변환하는 과정을 거쳐야만 `a / b`가 가능해집니다.
* `map(함수, 반복가능한객체)` 형태를 가집니다.
* 리스트 안에 들어있는 각각의 문자열들을 하나씩 꺼내 `float` 클래스에 집어넣어 한꺼번에 실수로 형변환을 시켜줍니다.
* 파이썬은 기본적으로 부동소수점 방식을 사용하기 때문에 소수점 표기를 따로 하지 않아도 정밀 출력이 가능합니다.
* 만약, 소수점을 표기해야하는 상황이 오면 아래 코드를 참고하시면 됩니다.
  ```python
  #f-string활용 소수점
    print(f"{a / b : .2f}")
   
  #ronud함수를 이용한 소수점
  #round(숫자, 자릿수)
    print(round(a / b, 2))
  ```
