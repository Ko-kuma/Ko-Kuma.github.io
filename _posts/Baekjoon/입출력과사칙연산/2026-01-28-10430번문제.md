## 문제 설명
(A+B)%C는 ((A%C) + (B%C))%C 와 같을까?

(A×B)%C는 ((A%C) × (B%C))%C 와 같을까?

세 수 A, B, C가 주어졌을 때, 위의 네 가지 값을 구하는 프로그램을 작성하시오.
## 문제 풀이
[input], [map], [int형변환], [.split()]를 활용하여 나머지를 구하면 되겠습니다.
```python
A, B, C = map(int, input().split())

print((A + B) % C)
print(((A % C) + (B % C)) % C)
print((A * B) % C)
print(((A % C) * (B % C)) % C)
```
f 포멧팅을 사용하지 않는 이유는 아래 코드를 보면 한줄로 입력되서 코드는 짧아 지지만 읽기 불편하고 어지럽다는 느낌을 받는 지저분한 코드가 됩니다.무조건, 짧은 코드가 좋은 코드가 아닌 가독성 좋은 읽기 편한 코드가 좋은 코드이기 때문에 꼭 f-string을 사용 할 필요는 없습니다. 상황에 따라 맞는 방식을 사용하면 됩니다.
```python
print(f"{(A+B)%C}\n{((A%C)+(B%C))%C}\n{(A*B)%C}\n{((A%C)*(B%C))%C}")
```


## 코드 설명
* map(int, input().split()): 세 개의 정수를 공백 기준으로 입력받아 각각 A, B, C 변수에 저장합니다.

* 나머지 연산자 %: 파이썬에서 %는 나눈 뒤의 나머지를 구하는 연산자입니다.

* 연산자 우선순위와 괄호: 파이썬에서 % 연산은 +, - 보다 우선순위가 높습니다.
    따라서 (A%C + B%C)%C라고 쓰면 괄호가 꼬여 의도치 않은 결과가 나올 수 있으므로, 공식에 맞게 괄호를 정확히 배치하는 것이 이 문제의 핵심입니다.s
