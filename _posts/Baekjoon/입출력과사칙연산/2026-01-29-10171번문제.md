## 문제 설명
고양이를 출력하시오.
## 문제 풀이
[print] 및 [/],["],[''']를 사용해서 해결 하면 됩니다
```python
print('''\\    /\\
 )  ( ')
(  /  )
 \\(__)|''')
```

## 코드 설명
* 삼중 따옴표('''): 여러 줄에 걸친 문자열을 출력할 때 사용합니다. 매 줄마다 print를 쓰지 않아도 되어 가독성이 좋아집니다.

* 백슬래시(\) 처리: 파이썬에서 \는 특수 기능을 수행하는 신호로 쓰입니다. 따라서 화면에 문자 그대로의 \를 출력하고 싶을 때는 \\와 같이 두 번 연속으로 써주어야 합니다.

* \\ /\\ : 첫 번째 \를 출력하기 위해 \\ 사용.
