## 문제 설명
꼬마 정민이는 이제 A + B 정도는 쉽게 계산할 수 있다. 이제 A + B + C를 계산할 차례이다!

## 문제 풀이
[map],[int],[input],[split]함수를 활용하여 문제를 풀면 됩니다.

```python
A, B, C = map(int, input().split())
print(A+B+C)
```

## 코드 설명
* input().split(): 입력받은 문자열을 공백(Space) 기준으로 쪼개어 리스트 형태로 만듭니다.

* map(int, ...): 쪼개진 문자열 리스트의 각 요소들을 정수형(Integer)으로 한꺼번에 변환합니다.

* A, B, C = ...: 변환된 세 개의 값을 각각의 변수에 할당(Unpacking)합니다.

* print(A+B+C): 세 수의 합을 출력합니다.
