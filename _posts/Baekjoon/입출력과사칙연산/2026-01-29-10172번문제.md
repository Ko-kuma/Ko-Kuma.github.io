## 문제 설명
개를 출력하시오.
## 문제 풀이
[print] 및 [r],[/],["],[''']를 사용해서 해결 하면 됩니다
```python
print(r'''|\_/|
|q p|   /}
( 0 )"""\
|"^"`    |
||_/=\\__|''')
```
raw string에 대해서는 아직 배우지 않거나 찾아보지 않으면 모르는 지식이 나와 헷갈릴 수 있습니다. 
그럴 때는 고양이 문제를 풀었듯이 풀면 되겠습니다(고양이 문제도 raw string으로 풀 수 있음.)
```python
print('''|\\_/|
|q p|   /}
( 0 )"""\\
|"^"`    |
||_/=\\\\__|''')
```
## 코드 설명
* Raw String (r''') 방식
    * 문자열 앞에 r을 붙이면 파이썬은 내부의 백슬래시(\)를 특수 기호로 해석하지 않고 보이는 그대로 출력합니다.
    덕분에 \\를 여러 번 쓸 필요 없이 눈에 보이는 모양 그대로 코드를 짤 수 있어 편리합니다.

* 일반 문자열 (''') 방식
    * 파이썬의 기본 규칙인 **이스케이프 시퀀스(Escape Sequence)** 를 따릅니다.
    \ 하나를 출력하기 위해서는 \\를, \\ 두 개를 출력하기 위해서는 \\\\를 입력해야 합니다.
