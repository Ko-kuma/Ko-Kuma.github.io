## 문제 설명
두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

## 문제 풀이
[int형변환],[for],[range]를 활용한 문제 풀이
```python
T = int(input())
for i in range(T):
    A, B = map(int, input().split())
    print(A + B)
```

## 코드 설명
* T = int(input()): 사용자로부터 앞으로 몇 번의 덧셈을 할 것인지(테스트 케이스 개수) 입력받습니다.
[쉽게 말해 몇번 반복하겠다]라는 의미입니다.

* for i in range(T): :
range(T)는 0부터 T-1까지, 총 T번의 숫자를 생성합니다.

* 변수 i는 여기서 단순히 **'몇 번째 반복인가'**를 세는 카운터 역할을 합니다.

* 반복문 내부의 입력: A, B = map(int, input().split())가 루프 안에 있기 때문에, 반복이 돌아갈 때마다 매번 새로운 숫자를 입력받을 수 있습니다.
