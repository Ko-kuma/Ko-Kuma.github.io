## 문제 설명
n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.



## 문제 풀이
[int형변환],[for],[range]를 활용한 문제 풀이
```python
n = int(input())
total = 0
for i in range(1, n + 1):
    total += i
print(total)

```

## 코드 설명
* total = 0: 합계를 저장할 변수를 초기화합니다. 반복문이 시작되기 전에 반드시 0으로 비워두어야 정확한 합을 구할 수 있습니다.

* for i in range(1, n + 1): :1부터 n까지 더해야 하므로, 끝값에 1을 더한 n + 1을 범위로 설정했습니다.i는 루프가 돌 때마다 1, 2, 3... n으로 변합니다.
* total += i: 이 연산은 total = total + i와 같습니다.만약 n=3이라면:
**1회차:** total(0) + i(1) $\rightarrow$ total은 1이 됨
**2회차:** total(1) + i(2) $\rightarrow$ total은 3이 됨
**3회차:**  total(3) + i(3) $\rightarrow$ total은 6이 됨

* print(total): 반복문이 모두 종료된 후(for문 밖) 최종 합계만 출력합니다.
