## 문제 설명
준원이는 저번 주에 살면서 처음으로 코스트코를 가 봤다. 정말 멋졌다. 그런데, 몇 개 담지도 않았는데 수상하게 높은 금액이 나오는 것이다! 준원이는 영수증을 보면서 정확하게 계산된 것이 맞는지 확인해보려 한다.

영수증에 적힌,

구매한 각 물건의 가격과 개수
구매한 물건들의 총 금액
을 보고, 구매한 물건의 가격과 개수로 계산한 총 금액이 영수증에 적힌 총 금액과 일치하는지 검사해보자.

## 문제 풀이
```python
X = int(input())
N = int(input())
total = 0
for i in range(N):
    a, b = map(int, input().split())
    total += a * b 
if X == total:
    print("Yes")
else:
    print("No")
```
## 코드 설명
* X: 영수증에 적힌 총금액입니다.

* N: 구매한 물건의 종류 수입니다.

* total = 0: 우리가 직접 계산해서 담을 '장바구니' 변수를 0으로 초기화합니다.

* 반복 계산 (for i in range(N):): 물건의 종류(N)만큼 루프를 돌며 각 물건의 가격(a)과 개수(b)를 입력받습니다.
total += a * b: 각 물건의 합계(가격 * 개수)를 기존 total 값에 계속 누적해서 더해줍니다.반복문이 완전히 끝난 후, 영수증 금액(X)과 우리가 계산한 금액(total)이 같은지 비교하여 결과를 출력합니다.
