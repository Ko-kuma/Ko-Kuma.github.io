## 문제 설명
N을 입력받은 뒤, 구구단 N단을 출력하는 프로그램을 작성하시오. 출력 형식에 맞춰서 출력하면 된다.

## 문제 풀이
[int형변환],[for],[range]를 활용한 문제 풀이
```python
A = int(input())
for i in range(1, 10):
    print(f"{A} * {i} = {A * i}")
```

## 코드 설명
* for i in range(1, 10): : range(시작값, 끝값) 함수는 시작값부터 끝값 바로 전까지의 숫자를 만들어냅니다.
즉, range(1, 10)은 1부터 9까지의 숫자를 하나씩 꺼내어 임시변수 i에 담아 반복합니다.

* f-string 활용: print(f"{A} * {i} = {A * i}")를 사용하여 변수와 수식을 문자열 안에 깔끔하게 삽입합니다.
