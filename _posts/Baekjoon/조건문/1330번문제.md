## 문제 설명
두 정수 A와 B가 주어졌을 때, A와 B를 비교하는 프로그램을 작성하시오.

## 문제 풀이
[if], [elif], [else]문을 활용하여서 문제를 풀면 되겠습니다.
```python
A, B = map(int, input().split())
if A > B :
    print(">")
elif A < B :
    print("<")
else :
    print("==")
```
아니면 one-line 조건문 형식으로 한 줄에 if문을 표현 가능
```python
print(f">" if A > B else f"<" if A < B else f"==")
```
## 코드 설명
* if 문: 첫 번째 조건인 A > B가 참(True)인지 확인합니다.

* elif 문: 앞의 조건이 거짓일 때, 두 번째 조건인 A < B를 확인합니다. else if의 줄임말입니다.

* else 문: 위의 모든 조건이 거짓일 때(즉, A == B인 경우) 실행됩니다.

* 파이썬의 삼항 연산자는 [참일 때 값] if [조건] else [거짓일 때 값]의 형태를 가집니다. 조건이 여러 개일 때는 else 뒤에 다시 if를 붙여 중첩할 수 있습니다.
