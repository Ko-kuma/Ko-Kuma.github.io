## 문제 설명
연도가 주어졌을 때, 윤년이면 1, 아니면 0을 출력하는 프로그램을 작성하시오.

윤년은 연도가 4의 배수이면서, 100의 배수가 아닐 때 또는 400의 배수일 때이다.

예를 들어, 2012년은 4의 배수이면서 100의 배수가 아니라서 윤년이다. 1900년은 100의 배수이고 400의 배수는 아니기 때문에 윤년이 아니다. 하지만, 2000년은 400의 배수이기 때문에 윤년이다.
## 문제 풀이
[if], [else], [논리연산자], [산술연산자]를 활용한 문제 풀이

```python
year = int(input())

if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print(1)
else:
    print(0)
```

## 코드 설명
* year % 4 == 0: 연도가 4로 나누어떨어지는지(4의 배수인지) 확인합니다.

* year % 100 != 0: 100의 배수가 아닌지 확인합니다. (!=는 '같지 않다'는 의미입니다.)

* year % 400 == 0: 400의 배수인지 확인합니다.

* 이 조건들을 문제의 요구대로 결합하면 아래처럼 됩니다

* (4의 배수 and 100의 배수 아님) : 이 두 조건은 동시에 만족해야 하므로 and로 묶습니다.

* or (400의 배수) : 위의 덩어리 또는 400의 배수이면 윤년이므로 전체를 or로 연결합니다.
