## 문제 설명
시험 점수를 입력받아 90 ~ 100점은 A, 80 ~ 89점은 B, 70 ~ 79점은 C, 60 ~ 69점은 D, 나머지 점수는 F를 출력하는 프로그램을 작성하시오.

## 문제 풀이
[if], [elif], [else]문을 활용하여서 문제를 풀면 되겠습니다.
문제를 풀때는 점수의 범위를 잘 생각하고 [and]논리연산자도 활용해주면 됩니다.
```python
A = int(input())
if A >= 90 and A <= 100:
    print("A")
elif A >= 80 and A < 90:
    print("B")
elif A >= 70 and A < 80:
    print("C")
elif A >= 60 and A < 70:
    print("D")
else:
    print("F")
```
one-line 조건문 형식으로 한 줄로 삼항연산자를 사용하여 표현이 가능하지만 오히려 짧은 문제가 아니면 삼항 연산자를 사용해 one-line으로 표기하면 가독성이 떨어져서 실수할 확률이 높아집니다.

## 코드 설명
* 논리 연산자 and: 두 가지 조건(A >= 90 그리고 A <= 100)이 모두 참일 때만 실행되도록 설정했습니다.

* 파이썬의 비교 연산자 축약: 파이썬에서는 90 <= A <= 100과 같이 수학 공식처럼 범위를 한 번에 표현할 수 있습니다. 이는 다른 언어(C, Java 등)에서는 불가능한 파이썬만의 강력한 문법적 설탕(Syntactic Sugar)입니다.

* 조건의 순서: if문은 위에서부터 순차적으로 검사하므로, 90 <= A 조건을 통과하지 못한 값들만 다음 elif로 내려오게 됩니다.
